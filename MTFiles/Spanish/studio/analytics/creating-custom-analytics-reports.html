    Creación de informes personalizados de Analytics

    En este tema, aprenderá a crear informes personalizados mediante el módulo Analytics.
    Nota: La capacidad de crear informes personalizados en el módulo Analytics solo está disponible para los editores Enterprise y Professional.
     

    El módulo Analytics se puede utilizar para generar informes analíticos personalizados y ad-hoc que se pueden descargar en formatos de valores separados por comas (.csv) o Microsoft Excel (.xlsx). La ejecución de los informes también se puede programar con una frecuencia diaria, semanal o mensual.

    El Analytics API se utiliza para recuperar datos del sistema de análisis. El también se Analytics API puede utilizar para crear sus propios informes personalizados. Para obtener más información sobre la Analytics API, consulte la documentación de Video Cloud Analytics API.

    Para comenzar a crear informes personalizados, abra el módulo Analytics y haga clic en el vínculo Personalizado en la navegación izquierda.

    Nota: Es posible que observe que el número de vistas de vídeo para un día determinado varía con el tiempo. Para obtener una explicación sobre por qué puede suceder esto, consulte las FAQ de Analytics.
    Nota: Los informes que devuelvan más de un millón de filas no se cargarán.
    Nota: Las solicitudes de informes personalizados simultáneas dentro de la misma cuenta se ejecutarán en serie. Si los informes tienen rangos de fechas grandes y se seleccionan muchos campos, puede haber un retraso antes de que se devuelvan todos los resultados. Una buena alternativa es programar informes grandes.
    Nota: la duración del vídeo se muestra en segundos.

    Para crear un informe personalizado, lleve a cabo los siguientes pasos:

    1. Seleccionar cuentas
    2. Seleccione un tipo de informe
    3. Seleccionar grupos de datos
    4. Agregar un filtro de informe
    5. Seleccionar campos de datos
    6. Vista previa del informe
    7. Configurar opciones de entrega
    8. Solicitar el informe

    Zona horaria de los informes

    En los datos de los informes personalizados, las fechas corresponden a la zona horaria de la cuenta desde la que se genera el informe. La zona horaria de informes se puede configurar desde la página principal de Video Cloud Studio yendo a Configuración de > cuenta ADMIN Zona horaria > predeterminada de informes. Recuerde que solo los administradores de cuentas pueden acceder a Ajustes de la cuenta.

    La zona horaria de generación de informes se utiliza para calcular los comienzos de los días reflejados en los informes. La modificación del ajuste de zona horaria para informes solo afecta a los datos desde el momento de efectuarse ese cambio, es decir, que no se aplica con carácter retroactivo. Si modifica este ajuste, puede que en los datos resultantes aparezca un tramo plano o un pico en los límites del día en que se efectuó esa modificación. Puede que la modificación de este ajuste no entre en vigor inmediatamente y que, por tanto, los datos de análisis sigan correspondiendo al ajuste anterior hasta que el sistema procese los cambios.

    Tenga en cuenta que se aplica la zona horaria de informes predeterminada solamente al módulo Analytics y no a Live Analytics, que técnicamente es un servicio separado por diseño

    Seleccionar las cuentas

    Si tiene acceso a varias cuentas de Video Cloud, puede incluir datos de la cuenta actual de Video Cloud o seleccionar varias cuentas.

    Si crea un informe que abarque todas las cuentas y estas utilizan diferentes zonas horarias, se aplicará a los datos la zona horaria definida en aquella cuenta.

    Seleccionar un tipo de informe

    Seleccionar el tipo de informe:

    • Una sola vez- El informe se ejecutará una vez, inmediatamente
    • Recurrente: el informe se puede programar para que se ejecute diariamente, semanalmente o mensualmente.
      Nota: Los informes recurrentes se ejecutan después de que se haya llevado a cabo la conciliación de análisis, normalmente de 3 a 5 días después del final de un período de informe.

    Informe ocasional

    Si selecciona Ocasional, el informe se ejecutará una única vez. Aparecerá un control de calendario para que usted pueda especificar el período que deba cubrir el análisis de datos.

    Consejo: En primer lugar, seleccione un intervalo de fechas más pequeño para comprobar si el informe contiene los datos que busca. Cuando haya comprobado los datos del informe, amplíe el intervalo de fechas según convenga.

    Informe recurrente

    Si selecciona Recurrente, el informe se ejecutará periódicamente. Seleccione la Frecuencia del informe:

    • Diario - El informe cubre un día entero
    • Semanal - El informe cubre una semana entera, con una semana definida como de lunes a domingo
    • Mensual - Los informes cubren un mes entero
    • Las cuentas institucionales se limitan a 20 informes periódicos, las cuentas del cuadro orgánico se limitan a 5
    • Los informes recurrentes quedan asociados a un usuario y están disponibles en todas las cuentas a las que el usuario tiene acceso
    • Los informes recurrentes se ejecutan después de que se haya llevado a cabo la conciliación analítica, normalmente de 3 a 5 días después del final de un período de informe

    Selección de grupos de datos

    Los grupos de datos determinan los campos de datos que se pueden incluir en el informe. Los campos de datos que aparecen en la sección Campos de datos se basan en los grupos de datos seleccionados. Aunque se pueden seleccionar múltiples grupos de datos, algunas combinaciones no son válidas. A medida que vaya seleccionando grupos de datos, la lista de grupos de datos disponibles se irá actualizando y solo mostrará los que se puedan seleccionar en cada nuevo contexto.

    Nota: El grupo de datos Fecha no se puede combinar con ninguna otra dimensión.
    Nota: Para ver una lista de campos de datos que se devuelven con cada dimensión, consulte este documento.

    Grupos de datos y campos

    Al seleccionar un grupo de datos, se pueden seleccionar los campos de datos que se pueden devolver. Los campos que están en gris no se pueden devolver para ese grupo de datos.

    Para añadir un grupo de datos, haga clic en el botón . Para eliminar un grupo, haga clic en .

    Añadir un filtro al informe

    Se pueden añadir filtros para acotar la cantidad de datos generados. Se aplicará la operación AND cuando se utilicen múltiples (dos o más) filtros. En el siguiente ejemplo, se devolverán los vídeos etiquetados como elementos esenciales de video cloud Y que tengan el campo personalizado Product=Video Cloud .

    Nota: En su informe sólo aparecerá un filtro cuando también haya seleccionado el grupo de datos correspondiente. Por ejemplo, si desea filtrar la etiqueta de vídeo, debe seleccionar la dimensión Vídeo.

    Para añadir un filtro, haga clic en el botón . Para eliminar un filtro, haga clic en . Puede utilizar como filtros los valores siguientes.

    • Vídeo - Uno o más ID de vídeo activos en la cuenta
    • Etiqueta de vídeo - Una o más etiquetas de vídeo
      • Al ingresar «servicios financieros» (con las comillas) se devolverán los vídeos etiquetados como servicios financieros.
      • Cuando se utilizan varias etiquetas, se devolverán los vídeos con TODAS las etiquetas. Al ingresar «servicios financieros», la contabilidad devolverá todos los videos que están etiquetados servicios financieros y contabilidad.
      • Nota: el mecanismo utilizado para devolver vídeos por etiqueta no tiene opciones de ordenación o paginación.
    • Campos personalizados : uno o más campos personalizados en la cuenta
      • Al filtrar campos personalizados y seleccionar Seleccionar todo , la lista de campos personalizados incluirá los campos personalizados de todas las cuentas. A medida que vaya seleccionando campos personalizados, la lista se irá filtrando para mostrar solo los campos personalizados comunes a todas las cuentas.
    • Jugador : el valor del filtro variará según el contexto que esté buscando:
      • Brightcove Players - Introduzca la URL del reproductor en este formato: players.brightcove.com/account_id/PlayerID_default
      • Reproductores de Brightcove utilizados en aplicaciones SDK : utilice la propiedad de destino de la aplicación. El valor predeterminado para la propiedad de destino es bcsdk:// seguido del identificador del paquete de la aplicación como se indica en el archivo info.plist de la aplicación.
    • Tipo de origen : uno de los siguientes valores:
      • Directo (marcador, escrito como URL o enlace en un correo electrónico)
      • Referencia (enlace directo desde otra página)
      • Búsqueda orgánica (búsqueda normal utilizando algún motor de búsqueda)
      • Búsqueda de pago (haciendo clic en un enlace patrocinado en los resultados de búsqueda)
      • Búsqueda segura (búsqueda de un espectador con sesión iniciada)
    • Tipo de dispositivo : cualquiera de los siguientes valores:
      • Móvil
      • Tableta
      • Televisor conectado/Consola de videojuegos/Dispositivo acompañante
      • Ordenador personal

    Seleccionar los campos de datos

    El apartado Campos de datos permite seleccionar los valores de los datos que se incluirán en el informe. La lista de campos de datos que aparecerán depende de las dimensiones que se seleccionen.

    Si la cuenta seleccionada está configurada con campos personalizados, también se pueden seleccionar. Si el informe incluye todas las cuentas, se mostrarán todos los campos personalizados de esas cuentas.

    Nota: Para los datos de utilización, solo informamos sobre segundos en vivo transmitidos, no ancho de banda en vivo.

    Previsualizar el informe

    La sección Vista previa del informe muestra las columnas de datos que se han seleccionado, rellenadas con datos de muestra. Haga clic en un icono de papelera para eliminar ese elemento de datos del informe.

    Nota: Los datos que se muestran en la vista previa del informe son datos de ejemplo, no datos reales de su cuenta de Video Cloud. Además, el orden de ordenación de la vista previa no afecta a cómo se ordenarán los datos en el informe que descargue. Los datos del informe descargado se pueden ordenar utilizando una aplicación de hoja de cálculo de terceros.

    Configurar las opciones de entrega

    Comience por asignar al informe un nombre lo más autoexplicativo posible. Esto es especialmente importante para informes recurrentes.

    El apartado Formato permite seleccionar el formato de los datos del informe que se descarguen.

    • CSV - valores separados por comas
    • XLSX - Microsoft Excel

    Puede utilizar notificaciones de correo electrónico para avisarse a sí mismo/a y a otros usuarios de su organización cuando el informe se haya generado. Al activar la opción Avisarme por correo electrónico cuando mi informe esté listo se generará un aviso por correo electrónico cuando el informe esté listo. Se puede notificar a otras personas incluyendo sus respectivas direcciones de correo electrónico en el campo de texto Notificar a otros. Cuando el informe esté listo, recibirán por correo electrónico un enlace al informe.

    El enlace para descargar el informe es válido durante 90 días.

    Solicitar el informe

    Haga clic en el Solicitar informe (para informes únicos) o Informe de programación (para informes periódicos) para ejecutar el informe. Si se trata de un informe ocasional, se descargará tan pronto como se haya generado. Los informes recurrentes se envían a la cola de informes y se ejecutan tan pronto como están listos los datos.

    Visualizar informes programados

    Para ver todos los informes recurrentes, haga clic en la opción Informes programados del panel de navegación izquierdo. Para obtener más información sobre cómo trabajar con informes programados, consulte Gestión de informes programados.
