    如何在分析模組中搜尋影片

    在本主題中，您將學習如何使用 Analytics (分析) 模組中提供的「快速尋找」功能來搜尋特定影片。

    使用快速尋找

    分析模組中的「快速尋找」功能可用來搜尋特定影片。輸入搜尋條件後，「快速尋找」會搜尋影片名稱、簡短說明、完整說明、標籤、視訊 ID 以及您可能擁有的任何自訂文字欄位。匹配列表將顯示在搜索字段下方。

    注意已刪除無法通過搜索功能搜索視頻，但我們會對其進行分析。您可以藉由展開/瀏覽效能報告，並使用瀏覽器搜尋功能，或是下載報表並在試算表應用程式中進行搜尋，即可找到這些項目。

    按一下符合項目的連結將會開啟「內容中」的報表。舉例來說，如果您正在查看帳戶的參與，然後使用 [快速尋找] 找出特定影片，按一下影片的連結就會開啟參與報表。如果您正在查看 [績效] 報表，搜尋並按一下連結將會開啟 [績效報告]。

    基本搜尋

    如果您提供沒有限定元的搜尋值，則會針對下列欄位執行文字搜尋：

    • 名字
    • 簡短描述
    • 完整描述
    • 標籤
    • 視訊 ID
    • 參考 ID
    • 所有自訂欄位

    如果你想匹配多個單詞，只需將術語包裝在引號中。

    進階搜尋

    Google Analytics（分析）模塊還提供了以下功能搜索特定字段，根據日期搜索和包含或排除特定條款。

    搜尋特定欄位

    若要搜尋指定值的特定欄位，請使用下列語法：

    • field:value to search

    支援下列欄位：

    欄位 支援的值
    name 字串或以括弧括住的字串
    text 字符串或帶引號的字符串（值name，short description，long description）
    tags 字串或以括弧括住的字串
    reference_id 字串或引號字串
    state ACTIVE，INACTIVE，DELETED，PENDING
    updated_at 請參閱搜尋特定日期格式
    created_at 請參閱搜尋特定日期格式
    published_at 請參閱搜尋特定日期格式
    complete true或false

    如果你想做一個完全匹配或匹配多個單詞，只需將術語包裝在引號中：

    • "live"
    • "live streaming"

    這也適用於搜索特定字段：

    • name:"live"
    • name:"live streaming"

    搜尋特定日期

    日期相關的搜尋欄位採用日期範圍值。支援指定日期的多種格式：

    格式 意思
    2013-08-01 日期 (年月)-日期格式
    下午 4 點 UTC 格式的時間
    -1D 當前時間減去1天（請參見相對日期）
    +1d 當前時間加上1天（請參見相對日期）
     

    相對日期

    對於相對日期，我們支持一個方向（+ 或-），後跟一個數字，後跟一個持續時間（或其縮寫）。法定持續時間為：分鐘 (m)、小時 (h)、天 (d)

    例如：

    • -1day
    • -2days
    • +4hours
    • -60minutes
    • -1d
    • -2d
    • +4h
    • -60m

    日期範圍

    如果您要搜尋的欄位是日期，您可以指定要搜尋的日期範圍：

    • updated_at:2013-08-01T00:00:00Z..2013-10-08T00:00:00Z

    這將搜索所有帶有updated_at值在2013年8月1日至2013年10月8日之間。

    開放結束範圍

    如果您想要比對指定時間之前的所有日期，或者比對自指定時間以來的所有日期，請留下範圍的一個結尾。例如，若要搜尋過去 2 天內修改的所有影片：

    • q=updated_at:-2days..

    搜尋所有在 2013 年 8 月 11 日之前修改的影片：

    • q=updated_at:..2013-08-11T00:00:00Z

     

    包括及排除特定條款

    您可以根據需要標記一個字詞（返回的視頻必須匹配）或禁止（必須返回視頻）不比賽）。這由+要么-緊接該術語之前簽名：

    範例 意思
    +animals 影片必須包括“動物”一詞name，short description， 要么long description
    -animals 視頻必須不包括“動物”一詞name，short description， 要么long description
    +name:animals 影片必須在名稱中包含“動物”一詞
    -name:animals 視頻必須不在名稱中包含“動物”一詞
