頁面內容

    視訊雲端詞彙

    視訊雲端和網際網路視訊的重要術語指南。

    | -A- | -B- | -C | -D- |-E-|-F-|-G- |-我- | -J- | -K- | -L | -M- | -N- | -O- | -P- | -Q-| - R -|-T-| - U | - V -Y- | -Z- |

    一個

    作用中媒體
    帳戶中使用中的影片數目。請注意，這些數據不是即時的-它是從您的視頻庫每天同步一次。另請注意，如果您正在查看「即時」分析數據（不到 3 天），則此數據可能不準確。
    分析 API
    REST 型應用程式開發介面可讓開發人員直接擷取 JSON 格式的 Video Cloud 影片的分析指標。Analytics API提供視訊雲端工作室中的分析使用者介面的資料。
    存取權杖
    暫時的 OAuth2 權杖，允許存取 REST API，例如 CMS API、分析 API 和動態內嵌 API。
    合作夥伴
    網頁發行商誰集團視頻雲播放器。
    廣告請求
    從視訊雲端播放器向廣告伺服器提出的要求 (呼叫)。廣告請求是 HTTP 呼叫，其中包含傳遞資訊的索引鍵/值配對，協助廣告伺服器針對相關內容定位適當的廣告。並非每個廣告請求都會傳回廣告。
    分析 API
    的Analytics API允許您以編程方式獲取Video Cloud帳戶和視頻的分析數據。
    蘋果 HLS
    蘋果 HTTP 直播串流，由蘋果實施的一種協議，用於提供視頻點播。Apple HLS 將視頻分成一系列小文件下載，每次下載都會使用 HTTP 加載一個短片的視頻。請注意，Apple HTTP Live Streaming與無關直播視頻。
    長寬比
    視訊影格的寬度與高度的比例。4:3 適用於標準系統，高畫質電視則為 16:9。

    B

    頻寬
    音訊或視訊設備可編碼或解碼的訊號頻率範圍。視訊使用的頻率高於音訊，因此需要更寬廣的頻寬。
    部落格
    一个由定期捐款组成的网络出版物，通常按时间相反的順序排列。
    布萊特灣玩家
    Brightcove 播放器是一個高性能的跨平台 HTML5 第一視頻播放器，快速載入，跨桌面和行動平台提供高品質的自適應位元率 HLS 視頻，並與您最喜愛的廣告和分析系統整合。
    布萊特灣玩家 API
    布萊特灣播放器 API 可以用來以程式設計方式控制布萊特灣播放器的外觀和行為。
    寬頻視訊頻道
    透過網際網路分發的相關影片集合。
    保險桿
    可在全長視訊之前或播放程式載入時出現的短視訊片段。

    C

    CMS 應用程式
    的CMS API提供對視頻元數據的未緩存讀取和寫入訪問。
    轉碼器
    用於以數位形式擷取類比視訊或音訊的演算法。與之比較容器。
    容器
    視訊容器是定義視訊中繼資料與視訊和音訊資料一起儲存的格式。它不定義數據的編碼或壓縮。例如，MP4 可以是 H.264 檔案的容器。
    內容傳遞網路 (CDN)
    提供網路伺服器的服務，透明地將內容 (特別是大型媒體內容) 傳遞給使用者。兩個突出的 CDN 是阿卡邁和聚光燈。
    內容目標設定
    內容目標是用於根據有關該訪問者的信息顯示不同的內容給不同的訪問者的過程。
    CSS
    級聯樣式表是用於控制 HTML 頁面的設計的標準機制。CSS 是用來風格布萊特灣球員。
    提示點
    在影片的特定點插入隱藏標記，可用來觸發外部事件，例如廣告中請求或隱藏式字幕。您可以使用「媒體」模組設定提示點。

    D

    破折號 (雙向破折號)
    透過 HTTP 動態調適性串流 (也稱為 MPEG-DASH) 是一種調適性位元速率串流技術，可透過從傳統 HTTP 網頁伺服器傳遞的網際網路進行高品質的媒體內容串流。如同蘋果的HTTP實時流（HLS）解決方案中，MPEG-DASH通過將內容分成一系列小的基於HTTP的文件段來工作。
    網域限制
    一種基本的內容安全性形式，防止播放程式在指定網域上運作。
    DRM
    數位版權管理是一種加密媒體檔案並封裝這些檔案的方法，以便只能根據原則金鑰將其解密。
    動態傳遞
    看到多比特流。
    動態內嵌 API
    動態內嵌 (DI) API 可用來指示視訊雲端擷取視訊資產並為其建立轉譯。的Dynamic Ingest API也可以用於提取字幕，章節或字幕的縮略圖和海報圖像或WebVTT文件。

    電子

    編碼
    將視訊和音訊資料轉換為可在播放器中播放的串流。
    加密
    編碼視訊的程序只有授權方才能檢視視訊。

    F

    影格速率
    顯示視訊片段的每秒影格數。

    G

    地理篩選
    對影片的地理限制，通常是因為版權擁有權的原因。

    H

    H.264 格式
    也被稱為 MPEG-4 AVC（高級視頻編碼）。用於透過網際網路傳送視訊的檔案格式。建議使用 H.264 格式，用於線上視訊傳送。H.264 檔案通常使用 MP4 容器，且副檔名為 .mp4。它提供比以前的格式更大的壓縮。
    HLS
    HTTP 直播流-由蘋果創建的視頻格式，使視頻在客戶端流的短片段提供。
    HTML
    (超文字標記語言) 一種標記語言，旨在建立具有超文字和其他資訊的網頁，以便在網頁瀏覽器中顯示。
    HTML5
    HTML5 是互聯網的核心技術標記語言，用於構建和呈現內容為萬維網。HTML5 是萬維網聯盟（W3C）的 HTML 標準的第五次修訂。

    我

    內嵌設定檔
    內嵌描述檔包含轉碼設定，當使用「上傳」模組 或 動態內嵌 API 上 傳影片時，將會使用這些轉碼設定 。
    內嵌設定檔 API
    內嵌描述檔 API 可讓您建立、擷取、更新和刪除視訊雲端帳戶的內嵌設定檔。內嵌描述檔會定義當您將視訊新增至 Video Cloud 時所建立的轉譯。

    J

    JavaScript
    指令碼語言以其在網站中的使用而聞名，雖然它也用來啟用內嵌在其他應用程式中的物件的指令碼存取。
    JPEG（聯合攝影專家組）
    用於數位影像的失真壓縮常用標準方法。JPEG 是 Brightcove 服務中使用的標準影像格式，適用於縮圖、影片靜態影像和背景影像。

    K

    金鑰/價值配對
    當玩家傳送廣告請求時，它會包含一或多個索引鍵/值配對 (例如類別 = 體育)，提供廣告伺服器關於廣告內容的資訊，並改善廣告目標鎖定。

    L

    信箱
    一種在不同縱橫比的螢幕上顯示寬螢幕視訊的方法，方法是在影像上方和下方留下空間 (通常是黑條)。
    標誌
    象徵性地代表公司品牌的專有圖形。
    標誌覆蓋
    顯示在視訊播放視窗中的小型透明影像。有時也稱為數位螢幕圖形 (DOG)、錯誤或浮水印。
    迴圈/迴圈
    播放清單中的視訊完成且播放清單開始在開始播放時。
    即時串流影片
    一種協議，通過直接推送捕獲和編碼的流來流式傳輸事件的視頻，這與分發編碼文件以進行流式傳輸相反視頻點播。

    M

    媒體庫
    視訊雲端帳戶中的影片、播放清單和其他資產的集合。
    中繼資料
    中繼資料是指您在影片定義中提供的說明、標記、類型及其他相關資訊。
    MPAA 電影評級系統
    由美國電影協會制定的一個系統，根據電影內容為電影評分。MPAA 評級制度（G、PG、R 等）是對潛在冒犯性內容進行分類的最受認可的制度，但通常僅限於電影行業使用，因為 MPAA 在每個個別評級上都有商標。
    多位元速率串流
    支持多個版本的功能，或者演繹，用於單個視頻，並能夠提供與觀看者的瀏覽器大小和帶寬最匹配的演示文稿。我們以前稱之為「動態交付。「
    靜音和解除靜音
    寫入和讀取各種音頻和視頻流容器格式。

    否

    原生播放程式 SDK
    Brightcove 原生播放器軟體開發套件提供建置 iOS 和 Android 應用程式所需的工具。

    O

    OAuth
    OAuth 是授權的開放標準。OAuth 代表資源所有者為客戶端應用程序提供對服務器資源的「安全委託訪問」。
    OAuth 應用程式介面
    OAuth API 是一個 RESTful API，允許您獲得授權令牌來訪問布萊特灣的其他 REST API。

    P

    播放
    視訊雲端播放器中的視訊媒體播放。
    播放 API
    播放 API 可在執行階段使用，存取行動應用程式或影片入口網站的視訊和播放清單資料。
    玩家管理 API

    播放程式管理 API 可用於：

    • 建立玩家
    • 檢視玩家的設定
    • 變更玩家的設定
    • 列出所有與發行者相關聯的玩家
    • 發佈播放程式，使其可供消費者使用
    玩家發佈代碼
    用於在網頁中嵌入 Brightcove 播放器的代碼「片段」。
    播放清單
    以特定順序組成群組的視訊集合，以便在 Video Cloud 播放器中播放。
    播放清單 API
    播放清單 API 用來以程式設計方式控制使用 JavaScript 的播放清單。
    政策 API
    Policy API用於創建或獲取可用於訪問播放API。原則金鑰會自動為布萊特灣玩家產生。
    海報影像
    在視訊播放前後，視訊播放器的播放區域中顯示的靜態影像。
    px
    像素的縮寫，用於CSS。

    Q

     
     

    R

    節目
    當您使用多位元速率串流時，您可以建立相同視訊的多個版本，每個版本都有不同的位元速率和尺寸。這些版本中的每一個都稱為轉譯。Video Cloud 多位元速率串流會根據播放器的尺寸和觀眾的連線速度，選擇最佳轉譯。
    參考 ID
    發佈商指派給影片或播放清單的唯一 ID。參考 ID (參考 ID) 與影片 ID 或播放清單 ID 不同，這些 ID 是由「影片雲端」產生的唯一 ID。參考 ID 可協調 Video Cloud 與其他服務 (例如發行者的內容管理系統) 之間的協調。
    RSS 摘要
    非常簡單的聯合是通知用戶更新的網站，博客或互聯網電視頻道中的內容的流行技術。

    S

    搜尋引擎
    Google 等專為協助尋找網路資訊而設計的程式。
    社交網站
    一個網站，如 Facebook 或 Twitter，它使其觀眾能夠彼此分享消息，媒體和其他信息的能力。
    直播中
    適用於網路和網際網路的低位元速率編碼格式。串流以編碼位元速率提供小片段的視訊，以符合使用者的連線速度，因此遠端檢視器可以播放音訊或視訊，而不需先下載整個視訊檔案即可播放音訊或視訊。

    T

    標籤
    組織內容的個別關鍵字或片語。Brightcove 支援使用標籤，作為發佈商整理和分類影片，以及建立播放清單的方式。
    縮圖
    與播放器、網站和視訊搜尋引擎中的影片清單中使用的影片相關聯的小型影像。
    權杖型驗證
    基於令牌的認證系統允許用戶輸入他們的用戶名和密碼，以獲取令牌，允許他們獲取特定的資源，而不需要他們的用戶名和密碼。一旦獲得了他們的令牌，用戶可以提供令牌，該令牌在給定時間段內提供對資源的訪問權限。
    電視家長指南
    由美國聯邦通訊委員會 (FCC) 建立的內容分級系統，用於根據內容的年齡適當性對電視節目進行分類。

    U

    URL
    統一資源定位器 (URL) 是識別資源及其在網際網路上的位置的方式 (例如 http://www.brightcove.com)。

    V

    可變位元速率
    一種壓縮方案，其中每個輸入材料單位可以壓縮成不同尺寸。例如，對於 MPEG-2 視訊，這意味著「更容易」的序列 (也就是沒有動作) 可以壓縮成非常小的尺寸，而「硬」序列 (具有大量動作和場景切割) 可以壓縮成更大的尺寸。
    視訊

    在影片雲端播放器中播放的單一影片標題。視訊是以視訊檔案為基礎，開始可以是許多支援的視訊檔案格式之一，但如果尚未使用其中一種格式，則會轉碼為 H.264 檔或 FLV 檔案。此外，視訊還可以包含中繼資料，例如：

    • 名稱
    • 簡短描述
    • 完整描述
    • 標籤
    • 視訊靜態影像
    • 海報影像
    • 保險槓
    • 標誌覆蓋
    視訊雲端工作室

    用於訪問和控制視頻雲服務的互聯網應用程序。該工作室包括一系列模塊，用於完成不同的任務：

    • 上載 -將視頻上傳到您的視頻雲帳戶
    • 媒體 -設置視頻屬性，使用文件夾組織視頻並發布視頻
    • 玩家們 -創建播放器並設置樣式，並為播放器生成發布代碼
    • 分析工具 -查看有關視頻和播放器性能的分析
    • 生活 -創建和直播現場直播
    • 畫廊 -為您的視頻內容創建視頻門戶
    • 聽眾 -將視頻分析從Video Cloud連接到營銷自動化平台
    • 社會的 -將視頻分發到YouTube，Facebook和Twitter
    視訊編輯工具
    在個人電腦上編輯視頻文件的軟件應用程序。
    視訊海報影像
    在視訊播放前後，視訊播放器的播放區域中顯示的靜態影像。
    VOD
    隨選視訊。一種通訊協定，允許透過串流或漸進式下載 (也就是隨選) 播放分散式視訊檔案。相對於直播視頻。

    W

    網路 VTT
    網路視訊文字軌跡格式。WebVTT 檔案提供視訊內容的隱藏式字幕或字幕，也提供文字視訊說明、內容導覽的章節，以及任何與音訊或視訊內容對齊時間的中繼資料形式。布萊灣玩家使用 WebVTT 格式的字幕。

    X

    XLSX
    微軟 Excel 文件。分析資料可以這種格式匯出。
