    逐步：視頻雲工作室發布工作流程

    在本主題中，您將使用 Video Cloud Studio 進行基本的視訊發佈工作流程。本主題提供了一組親手操作的步驟，您可以自行完成以探索 Video Cloud 的某些功能。

    該主題是視頻雲簡介按需培訓課程。該訓練中的實際操作練習是以本文件中的步驟為基礎。建議您觀看該訓練，然後使用本文件，以便您可以自行完成實際操作步驟。

    先決條件

    • Video Cloud帳戶-您可以註冊一個試用帳戶 這裡。
    • 您應該可以使用一些示例視頻，也可以 下載 一些樣本視頻
    注意：本主題中的步驟和螢幕擷取畫面會使用所提供的範例視訊。

    外框

    本主題將會針對單一影片和播放清單進行基本的視訊發佈程序。涵蓋下列主題：

    • 上傳視訊
    • 編輯視訊屬性
    • 創建播放器和样式
    • 將視頻發佈到網絡
    • 建立播放清單
    • 發布播放列表

    視訊雲端分析模組並未作為此快速入門的一部分。要了解視頻雲分析，請觀看 視頻雲分析簡介 按需培訓課程。

    將視訊上傳至視訊雲端

    在繼續之前，您應該有一些可以使用的示例視頻，或者下載樣本視頻。

    登入您的視訊雲端帳戶

    1. 在瀏覽器窗口中，導航到https://studio.brightcove.com。
    2. 輸入您的憑據，然後單擊登入。

    將本機影片上傳至視訊雲端

    1. 請點擊上載打開上傳模塊。
    2. 在「上傳」模組中，選取內嵌設定檔。
    3. 請點擊瀏覽影片並瀏覽並選擇您的視頻或提供的所有示例視頻。
    4. 該頁面將顯示所有視頻和上傳進度。

      上傳處於處理狀態時，請勿關閉或離開瀏覽器視窗。這樣做可能會導致上傳不完整。

      上傳完成後，會顯示「處理完成」狀態。這表示處理已完成，並且可以安全地從「上傳」模組瀏覽。

      注意：可以單擊編輯文件名旁邊的按鈕可編輯一些視頻元數據。下一個快速入門主題將涵蓋新增視訊中繼資料。

    將遠端視訊上傳至視訊雲端

    1. 在裡面上載模塊，單擊輸入網址。
    2. 輸入以下URL到Amazon S3帳戶上託管的文件。
      http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4
    3. 請點擊處理影片開始上傳。

    (選擇性) 使用行動裝置上傳影片

    1. 使用移動設備在以下位置打開Video Cloud Studio https://studio.brightcove.com。
    2. 打開上載模塊。
    3. 選取內嵌設定檔。
    4. 請點擊瀏覽影片。裝置上的相機膠卷應用程式應該會開啟。
    5. 選擇要上傳的影片。將開始上傳，並顯示上傳進度。

    編輯視頻屬性

    在此任務中，您將編輯視頻屬性。

    1. 在視訊雲端 Studio 中，按一下 [ 媒體 ] 以開啟 [媒體] 模組。
    2. 單擊標題為褐鵜鶘視頻，找到影片資訊部分，然後單擊編輯。設置簡短的介紹至Pelican on the rocks。
    3. 添加一個標籤的birds。
    4. 請點擊返回影片返回視頻列表。
    5. 單擊標題為http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4視頻，找到影片資訊部分，然後單擊編輯。設定下列屬性：
      • 名稱：Oystercatcher
      • 簡短的介紹：Oystercatcher on the beach
      • 標籤：birds

      請點擊保存完成後，單擊返回影片返回視頻列表。

    6. 編輯兩個山雀和啄木鳥視頻。標記視頻birds。
    7. （可選）編輯一些動物視頻並用標記視頻animals。

    建立和設定播放器樣式

    在這個任務中，您將創建一個玩家和風格。

    建立新玩家

    1. 在視訊雲端工作室中，按一下播放器以開啟播放器模組。
    2. 按一下「新播放器」。
    3. 輸入Custom Player輸入名稱，然後單擊保存。

    設定播放器樣式

    1. 點擊自訂播放器鏈接以打開播放器屬性。
    2. 確認玩家類型是單個視頻。
    3. 按一下左側導覽功能表中的「樣式」。
    4. 檢查一下顯示標題和描述。
    5. 選擇播放按鈕形狀和播放按鈕位置。
    6. 點擊主要顏色選擇器，並將顏色設置為紅色陰影。
    7. 確認顏色變更反映在預覽播放器中。
    8. 請點擊發布和嵌入>發布更改發布更改。

    將視訊發佈到網路

    在此任務中，您將發布視頻到網絡。

    發佈單一視訊

    1. 在視訊雲端 Studio 中，按一下 [ 媒體 ] 以開啟 [媒體] 模組。
    2. 點擊旁邊的複選框褐鵜鶘視頻，然後單擊發布和嵌入>發佈到Web。
      注意：如需其他發佈選項的相關資訊，請參閱對應的文件：
      • 電子郵件登陸頁面 -- 將視頻發佈到電子郵件
      • 社交網絡 -- 將視頻發佈到社交媒體平台
      • 畫廊頁內體驗 -- 將視頻發佈到頁內體驗
    3. 確認開啟 [發佈] 對話方塊。
    4. 在裡面選擇播放器下拉列表選擇自訂播放器。這個播放器是在早期部分建立的。
      注意：在所有視訊雲端帳戶中，預設會建立布萊特灣預設播放器。
    5. 按一下預覽播放器下方的 URL ，在新的瀏覽器標籤中開啟該網址。確認視訊出現且可以播放。
      注意：請點擊縮短生成一個縮短的URL。
    6. (選擇性) 將 URL 以電子郵件傳送至您的行動裝置。按一下連結並確認影片是否在您的行動裝置上播放。

    建立播放清單

    在這個任務中，您將創建一個手動和智能播放列表。

    建立手動播放清單

    1. 在視訊雲端 Studio 中，按一下 [ 媒體 ] 以開啟 [媒體] 模組。
    2. 在側邊導覽面板中，按一下「建立播放清單」連結。新增播放清單對話方塊隨即開啟。
    3. 將播放清單類型設定為「手動」。
    4. 將播放清單名稱設定為Favorites。
    5. 勾選「在側邊欄中顯示播放清單」，讓播放清單出現在側邊欄中。
    6. 按一下「儲存」。

    將視訊新增至手動播放清單

    1. 請點擊所有影片在側面導航面板中顯示該帳戶中的所有視頻。
    2. 按一下您要新增至播放清單中的影片旁邊的核取方塊。
    3. 請點擊更多>添加到播放列表。
    4. 選擇收藏夾播放列表，然後單擊加。
    5. 點擊收藏夾側面導航面板中的播放列表，並確認視頻已添加到播放列表中。
    6. 按一下某列並將其拖曳至清單中的新位置，重新排列視訊的順序。

    建立智慧型播放清單

    1. 在側邊導覽面板中，按一下「建立播放清單」連結。
    2. 將播放清單類型設定為「智慧型」。
    3. 將播放清單名稱設定為Birds。
    4. 勾選「在側邊欄中顯示播放清單」，讓播放清單出現在側邊欄中。
    5. 設置標籤到包含一個或多個然後輸入birds標籤。
    6. 按一下「儲存」。
    7. 點擊鳥類側面導航面板中的播放列表，並確認已將鳥類視頻添加到播放列表中。

    發佈播放清單

    在這個任務中，您將創建一個新的播放器，編輯播放器設置以支持播放列表，然後發布播放列表。

    編輯播放器設定以支援播放清單

    注意：如果選擇反應靈敏調整大小選項後，標準（iframe）代碼效果很好，並且播放器和播放列表都具有響應能力。如果您使用進階 (頁內嵌) 程式碼，播放程式將會回應，但播放清單不會回應。
    1. 在視訊雲端工作室中，按一下播放器以開啟播放器模組。
    2. 按一下「新播放器」。命名玩家Playlist Player然後點擊保存。
    3. 找到播放列表播放器並單擊播放器名稱以打開播放器屬性。
    4. 選擇一個玩家類型的播放清單。
    5. 設置長寬比至16：9。
    6. 設置播放器尺寸到寬度的640。
    7. 按一下「儲存」。
    8. 按一下左側導覽功能表中的「樣式」。
    9. 點擊播放清單類型的播放列表（垂直）。
    10. 按一下左側導覽功能表中的「播放」。
    11. 檢查一下選中時播放視頻。
    12. 檢查一下連續播放模式。
    13. 請點擊發布和嵌入>發布更改將更改發布給播放器。

    發佈播放清單

    1. 請點擊媒體打開媒體模塊。
    2. 請點擊所有播放清單在側面導航面板中以顯示所有播放列表。
    3. 點擊旁邊的複選框鳥類播放列表。
    4. 點擊發布和嵌入>發佈到Web。
    5. 選擇播放列表播放器。
    6. 點擊網址鏈接。玩家將會在新的瀏覽器分頁中開啟。確認播放清單播放程式出現，且播放清單中的影片可以播放。
    7. (選擇性) 將播放器網址以電子郵件傳送至您的行動裝置。按一下連結，確認播放清單播放器出現，並且播放清單中的影片可以播放。
