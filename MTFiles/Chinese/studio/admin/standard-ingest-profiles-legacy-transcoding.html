    標準內嵌設定檔以進行舊版轉碼

    在本主題中，您將瞭解使用舊式轉碼時所提供的標準內嵌設定檔的屬性。

    使用「上傳」模組上傳視訊或使用 DI API 建立動態內嵌請求時，您必須提供內嵌設定檔，以決定 Video Cloud 擷取程序所建立的轉譯數目和類型。本主題提供每個舊式內嵌設定檔的詳細資訊。如需動態傳遞設定檔的詳細資訊，請參閱動態傳遞的標準內嵌設定檔。

    注意：上傳品質較低 (位元速率較低) 來源內容並選取其中一個標準描述檔時，Video Cloud 只會產生等於或低於來源品質的轉譯。

    也可以創建自己的自定義攝取配置文件。如需資訊，請參閱管理內嵌設定檔。

    文件資料

    注意事項：
    • 此頁面是使用 API 的即時資料建立；完全載入可能需要一分鐘的時間。
    • 可能無法編輯標準設定檔-如果您想要修改版本，則需要建立自訂設定檔。
