    使用上傳模組上傳視訊

    本主題概述了用於將視頻上傳到Video Cloud帳戶的Upload模塊。
     

    「上傳」模組使用動態內嵌程序來擷取和轉碼視訊內容。當您選擇要上傳的檔案時，Video Cloud 會使用以提取為基礎的程序，從儲存位置提取內容，然後使用 ZenCoder 轉碼服務來轉碼內容。

    要訪問上傳模塊，請點擊上載在導航標題中。

    支持的視頻格式

    有關視頻格式和編解碼器以及Video Cloud支持哪些源視頻格式的信息，請參閱支持的視頻編解碼器和容器。

    選擇攝取配置文件

    在上傳視頻之前，請選擇一個攝取配置文件。攝取配置文件包含在對視頻進行轉碼時將使用的轉碼設置。可用攝取配置文件的列表將根據您的視頻雲帳戶的配置方式而有所不同。在下面的屏幕截圖中，CONTEXT AWARE ENCODING和DYNAMIC DELIVERY配置文件可用。

    注意：要了解有關動態投放的更多信息，請參見動態交付概述。
    注意：高效視頻編碼（HEVC）配置文件也可以作為付費附件使用。要了解有關HEVC的更多信息，請參見HEVC視頻。

    創建的視頻演示文稿基於所選的攝取配置文件。有關Brightcove提供的攝取配置文件的特定詳細信息，請參見用於動態交付和上下文感知編碼的標準攝取配置文件

    海報和縮略圖的圖像尺寸基於攝取配置文件中指定的圖像尺寸。

    注意：有關創建其他提取配置文件或為您的帳戶設置默認配置文件的信息，請參見管理攝取配置文件。

    上傳視訊

    注意：上傳影片之前，請先確定已選取適當的內嵌描述檔。

    您可以使用下列其中一種方法上傳影片：

    • 拖放 -將文件從文件系統拖到“上傳模塊”頁面。導航標題下的整個頁面將成為放置區域。當檔案被刪除時，上傳程序將開始。
    • 檔案選取對話方塊 -按一下「瀏覽視訊」以開啟檔案選取對話方塊，然後瀏覽並選取要上傳的檔案。
    • 遠端網址 -點擊輸入網址指定源文件的位置。源文件可以駐留在公共Dropbox文件夾或Amazon S3存儲位置上。請點擊處理影片等結束了。
    注意事項：
    • 沒有檔案大小限制。布萊特灣已經測試過大到 100GB 的上傳，沒有任何問題。
    • 上傳大文件時，Brightcove建議您設置上傳排隊設置為一次上傳一個文件。
    • 如果在上傳大文件時收到Studio超時錯誤，請嘗試將文件上傳到S3或其他遠程存儲位置，並使用Studio中的URL上傳選項。
    • 無法使用YouTube網址，因為它們不會生成指向視頻文件的直接鏈接。
    • 有關從Google雲端硬盤上傳文件的信息，請參見使用遠程URL上載視頻。
    • 從Dropbox上傳時，如果URL以“？dl = 0”結尾，則在文件上傳之前，您可能需要將其更改為“？dl = 1”。
    • 視訊檔案名稱 (包括副檔名) 不得超過 120 個單位元組 (60 個雙位元組) 字元。如果這樣做，視訊將會成功攝入，但您將無法在稍後重新轉碼。

    選取檔案後，檔案上傳就會開始。若要取消上傳，請按一下「取消上傳」。按一下「關閉」，即可從頁面中移除檔案項目。上傳處於處理狀態時，請勿關閉或離開瀏覽器視窗。這樣做可能會導致上傳不完整。

    上傳完成後，會顯示「處理完成」狀態。這表示處理已完成，並且可以安全地從「上傳」模組瀏覽。

    請點擊編輯查看攝取配置文件和視頻屬性的子集。如果視頻轉碼尚未開始，則可以更改攝取配置文件。可以編輯以下視頻屬性：

    • 狀態 -視頻狀態
      • 活性 -可以播放視頻
      • 不活躍 -無法播放視頻

      默認情況下，上載視頻時，除非缺少必填字段，否則狀態將設置為“有效”。如果缺少必填字段，則會顯示一條消息，其中必填字段以紅色突出顯示。遺失必填欄位的影片仍然可以上傳，但在所有必要欄位都有值之 前，這些影片仍會保持「 非作用中」狀態。有關使用必填字段的更多信息，請參見設置視頻的必填字段。

      注意：有一個帳戶級別設置，默認為不活動，可用於將所有上傳的視頻設置為非活動狀態。有關更多信息，請參見管理帳戶設置。
    • 名稱 -視頻標題
    • 簡短的介紹 -視頻的簡短說明
    • 標籤 -與視頻相關的標籤
    • 參考編號 -視頻參考ID
    • 視頻投影 - 選擇360度表示視頻是360度視頻。有關使用360度視頻的更多信息，請參閱處理360°視頻。
    • 啟用離線播放 -選中後，當使用通過設備SDK創建的應用程序時，可以將視頻保存在本地並脫機觀看。僅當該帳戶啟用了脫機播放時，才會顯示此選項。建議僅在使用時設置此選項進階上傳（當用戶必須手動開始上傳處理時）。在視頻上傳過程中設置此設置可能會導致問題。
    • 移至文件夾 -選擇要將視頻分配到的文件夾。有關使用文件夾的更多信息，請參見使用文件夾整理視頻。
    • 自定義字段 -將顯示已標記為必填項的所有自定義字段。

    要了解有關視頻屬性的更多信息，請參見使用媒體模塊編輯視頻屬性。

    請點擊保存保存所有更改並開始轉碼過程。

    使用高級上傳

    當。。。的時候進階上傳帳戶設置已啟用，視頻屬性面板也會顯示選擇按鈕，可將海報和縮略圖圖像與視頻一起上傳。請點擊選擇選擇海報和縮略圖，然後單擊處理開始轉碼。

    請注意，海報和縮略圖的圖像尺寸將基於攝取配置文件中指定的圖像尺寸。有關配置進階上傳帳戶設置，請參見管理帳戶設置。

    設置默認文件夾和標籤

    上傳視頻時，可以指定要上傳的視頻的文件夾和標籤。文件夾下拉列表將顯示在“媒體”模塊中創建的所有文件夾的列表。請點擊+創建文件夾創建新文件夾並將視頻分配到新文件夾。創建新文件夾時：

    • 名稱限制為 100 個字元
    • 資料夾名稱在帳號中必須是唯一的
    • 非法字元：/

    選擇文件夾後，所有上傳的視頻都將添加到該文件夾中。有關使用“媒體”模塊創建視頻並將視頻分配到文件夾的信息，請參閱使用文件夾整理視頻。

    標籤也可以在上傳時分配給視頻。輸入以逗號分隔的標籤列表，以分配給視頻。

    選擇文件夾或添加標籤後，單擊全部應用保存更改。請注意，文件夾和標籤將應用於所有上傳的視頻，正在上傳的視頻或上傳完成的位置，並且視頻仍顯示在“上傳”模塊中（解僱按鈕尚未被點擊）。

    從移動設備上傳視頻

    Media模塊完全用HTML5編寫，也可以在平板電腦和移動設備上使用。這使您可以使用移動設備拍攝視頻並將其上傳到您的Video Cloud帳戶。要從移動設備上傳內容，請打開“上傳”模塊，然後選擇一個攝取配置文件。當您單擊瀏覽影片，設備上的照片應用程序將打開。選擇要上傳的視頻，然後單擊完成。來自移動設備的視頻將開始上傳，並且進度將顯示在您的設備上。

    特殊字元

    如果您提供視頻雲獲取視頻的URL，並且要發送憑據作為URL的一部分，則需要百分比編碼某些保留的特殊字符：

    字元 百分比編碼
    ] %5B
    [ %5D
    ? %3F
    / %2F
    < %3C
    ~ %7E
    # %23
    ` %6D
    ！ %21
    @ %40
    $ %24
    % %25
    ^ %5E
    & %26
    * %2A
    ( %28
    ) %29
    + %2B
    = %3D
    } %7D
    | %7C
    ： %3A
    「 %22
    ; %3B
    ' %27
    ， %2C
    > %3E
    { %7B
    空間 %20

    例如：

    
    /* unencoded */
    ftp://user@example.com:pass!word@example.com/path/to/input.mp3

    /* encoded */
    ftp://user%40example.com:pass%21word@example.com/path/to/input.mp3

    常見的上傳錯誤

    請求時間與當前時間之間的差異太大。

    當機器上載視頻的系統時間設置不正確時，將發生此錯誤。這通常在更改時間後發生。確保您的機器設置為正確的時間。
