페이지 내용

    미디어 모듈을 사용하여 비디오에 캡션 추가

    이 항목에서는 미디어 모듈을 사용하여 비디오에 캡션을 추가하는 방법을 배웁니다.
     

    텍스트 트랙 속성은 비디오에 대한 캡션을 제공하도록 구성할 수 있습니다. 캡션은 액세스하려는 개인에게 추가 또는 해석 정보를 제공합니다. 청각 장애인과 난청자가 비디오 프레젠테이션의 오디오 부분에 액세스할 수 있도록 자막이 필요합니다. Brightcove 플레이어에서 비디오에 대한 캡션을 표시하려면 다음 작업을 완료해야합니다.

    • 캡션 파일 만들기
    • 텍스트 트랙 속성 구성
    • 스타일 캡션

    캡션 파일 만들기

    비디오에 캡션을 추가하려면 먼저 캡션 파일을 만들어야 합니다. Brightcove 플레이어는 WebVTT (Web Video Text Tracks Format), SRT, DFXP 및 SCC 형식을 지원합니다. WebVTT 파일 형식 (아래 참조) 은 일반 텍스트이며 시간 텍스트 데이터 다음에 원하는 캡션 또는 자막 텍스트가 포함됩니다.

    참고 사항

    • SRT, SCC 및 DFXP 캡션은 Dynamic Delivery 계정의 WebVTT로 자동 변환됩니다.
    • 캡션의 텍스트는 일본어 및 한국어와 같은 2 바이트 언어에 대해 올바르게 표시되도록 UTF-8로 인코딩되어야합니다.
    • 캡션은 HTML로 표시되기 때문에 내부의 모든 텍스트는< >대괄호는 태그로 해석되며 표시되지 않습니다.

    참고:

    • 타사에서 만든 캡션이 타임 스탬프로 캡션을 시작하는 경우를 보았습니다.

      01 : 00 : 00.000

      이렇게하면 캡션이한 시간비디오에. 캡션이 비디오와 올바르게 동기화되었는지 확인하려면 타임 스탬프가 00 : 00 : 00.000모든 타임 스탬프에서 01시간 오프셋.

    WebVTT 형식에 대한 자세한 내용은이 링크를참조하십시오. 지원해야 하는 언어마다 별도의 캡션 파일이 있어야 합니다. 자막 파일을 만드는 데 도움이 되는 타사 도구가 많이 있지만 브라이트코브에는 전사, 자막 및 번역 서비스를 제공하는 여러 파트너가 있습니다. Brightcove 파트너의 전체 목록을 보려면 다음을 방문하십시오. 파트너우리 웹 사이트의 섹션.

    텍스트 트랙 속성 구성

    캡션 파일이 생성되면 브라이트코브에서 업로드 및 호스팅하거나 인터넷에 액세스할 수 있는 URL에 상주할 수 있습니다.

    비디오의 텍스트 트랙 속성을보고 편집하려면 미디어 모듈을 열고 비디오 제목 링크를 클릭 한 다음텍스트 트랙부분. 텍스트 트랙 속성을 구성하려면 다음 단계를 따르십시오.

    1. 편집을클릭하거나 섹션을 두 번 클릭합니다.
    2. 텍스트 트랙 파일을 Video Cloud에 업로드할지 또는 외부 URL에서 호스팅할지 선택합니다.
      • 파일 업로드 - 찾아보기를클릭하고 파일 시스템에서 텍스트 트랙 파일을 선택합니다. 파일이 비디오 클라우드에 업로드됩니다.
      • URL 업로드 - 텍스트 트랙 파일이 있는 공용 URL을 입력한 다음업로드를누릅니다. 파일이 비디오 클라우드에 업로드됩니다.
      • 원격 URL - 텍스트 트랙 파일이 있는 공용 URL을 입력합니다. URL은 250자 미만이어야 합니다. 캡션은 원격 URL에서 로드됩니다.
        참고: 자체 서버에서 자막 파일을 호스팅하는 경우 파일이 브라이트코브 플레이어가 액세스할 수 있는 URL에 있는지 확인합니다. 캡션 파일을 호스팅하는 서버에는 CORS (교차 원본 리소스 공유) 가 활성화되어 있어야 합니다. CORS에 대한 자세한 내용은여기에서읽을 수 있습니다.
        참고: 괄호, 물음표, 따옴표, 공백 및 기타 특수 문자가 포함된 텍스트 트랙 파일 이름은 업로드 시 자동으로 실패합니다. 브라이트코브는 특수 문자 없이 파일 이름을 사용할 것을 권장합니다.
        참고: 계정에 공유 된 비디오의 텍스트 트랙은 미디어 모듈에서 편집 할 수 없습니다.
      참고: 그만큼기본값 (자동 표시) 옵션은 현재 Dynamic Delivery 타이틀에 영향을 미치지 않습니다.
    3. [ 언어 ] 필드에는 트랙 텍스트 데이터의언어를 지정하는 두 글자 언어 코드가포함되어야 합니다.
    4. 캡션 파일에 적합한레이블을입력합니다. 레이블은 플레이어 내부에 표시되며 뷰어가 언어를 선택할 수 있습니다. 레이블을 지정하지 않으면 언어 코드가 표시됩니다.
    5. 텍스트 트랙의종류를선택합니다. 옵션은 다음과 같습니다.
      • 자막 - 자막과 유사하지만 음향 효과 및 기타 오디오 정보도 포함할 수 있습니다.
      • 자막 - 대화의 전사 또는 번역
      • 설명 - 화면 판독기를 통해 비디오를 설명하는 별도의 텍스트 파일입니다.
      • 장 - 사용자가 비디오를 탐색하는 데 도움을주기 위해
      • 메타데이터 - 기본적으로 시청자에게 표시되지 않는 동영상에 대한 정보 및 콘텐츠 (JavaScript를 사용하여 원할 수도 있음)
    6. 텍스트 트랙 파일의MIME 유형을입력합니다. 선택기본비디오가 재생 될 때 캡션을 표시합니다 (Brightcove Players 6.x 이상).
      SRT 또는 DFXP 캡션을 업로드하는 경우 MIME 유형을 편집 할 수 없습니다.
    7. 트랙을기본값으로 설정하려면 기본값 (자동 표시)을 선택합니다. 대부분의 브라우저는 기본 텍스트 트랙을 먼저 표시하려고 시도하지만 언어 및 접근성 설정이 기본 설정보다 우선할 수 있습니다.
      참고: 기본 설정은 IE11에서 작동하지 않습니다.
    8. 텍스트 트랙 추가를클릭하여 텍스트 트랙 정보를 저장합니다.
    9. 텍스트 트랙 추가를클릭하여 다른 텍스트 트랙을 추가하거나저장을클릭하여 텍스트 트랙을 저장합니다.
      참고: 텍스트 트랙 속성에 대한 변경 사항이 게시 된 플레이어에 표시되는 데 최대 15 분이 걸릴 수 있습니다.

    기본값으로 지정된 텍스트 트랙이 있으면굵게표시됩니다. 텍스트 트랙을 삭제하려면 삭제 아이콘 ( ) 텍스트 트랙 옆에 있습니다.

    값은 트랙당default값이 아니라 동일한 언어의 모든 트랙에true대해 로 설정됩니다 ( srclang ).

    연결된 자막이 있는 플레이어가 게시되면 플레이어 크롬에CC버튼이 나타납니다. CC버튼을 클릭하여 캡션 언어를 선택하고 캡션을 표시합니다.

    캡션 속성 편집

    기존 캡션의 속성을 편집하려면편집하다텍스트 트랙 패널에서 편집 할 캡션 항목을 클릭합니다.

    캡션 스타일 지정

    캡션의 기본 모양과 느낌은 변경될 수 있으므로 캡션의 스타일을 지정할 수 있습니다. 캡션 스타일 지정에 대한 자세한 내용은스타일링 캡션 .

    제한 사항

    • 캡션은 다음 장치 유형 및 운영 체제에서 지원됩니다.
      • iOS 10+
      • 안드로이드 7
      • 윈도 인터넷 익스플로러
      • 데스크탑에서 최신 크롬, 사파리, 파이어폭스 브라우저
    • 캡션 길이에는 기술적인 제한이 없지만 전체 캡션을 표시하려면 플레이어가 그 많은 텍스트를 표시할 수 있을 만큼 커야 합니다.
    • Safari는 자막 스타일을 지정하는 기본 기능을 사용하기 때문에 Safari 브라우저 (iOS 및 macOS 모두) 의CC메뉴에서캡션 설정메뉴 옵션을 사용할 수 없습니다.
    • Studio에서 잘못된 WebVTT 파일을 수집하면 파일이 "처리"되지만 동영상에 추가되지는 않습니다. 해결 방법 : 다음을 사용하여 WebVTT 파일의 유효성을 검사 할 수 있습니다. 이 온라인 유효성 검사기 .
