페이지 내용

    성능 보고서 참조

    이 항목에서는 Analytics 모듈의 일부로 제공되는 성능 보고서에 대해 알아봅니다.
     

    분석 데이터는 네이티브 플레이어 SDK에서 제공하는 데이터를 포함하여 브라이트코브 플레이어가 자동으로 전송합니다. 당신이있는 경우아니 Brightcove Player를 사용하여 Video Cloud 비디오를 제공하려면 데이터 수집기로 데이터를 전송하는 데 사용중인 플레이어를 계측해야합니다. 참조개요 : 데이터 수집 API자세한 내용은 주제를 참조하십시오.

    실적 보고서를 보려면 분석 모듈을 열고공연왼쪽 탐색에서. 보고서에는 선택한 기간 동안 하루에 총 비디오 조회수를 보여 주는 그래프가 표시됩니다. 드롭다운 목록을 사용하여 다른 지표를 선택할 수 있습니다.

    • 비디오 조회수 -스트림이 시작될 때 녹화 된 비디오 재생 시작 횟수 (되감기 또는 다시 재생은 포함되지 않음) 개별 시청자의 척도가 아닙니다. 프리 롤 광고가있는 동영상의 경우 동영상 재생이 시작될 때까지 조회수가 기록되지 않습니다.
    • 비디오 노출수 - 비디오가 로드되어 상호 작용이 준비된 횟수
    • 재생 비율 - 로드된 동영상의 실제 재생된 비율 (동영상 조회수/동영상 노출 수)
    • 본 분 -동영상 시청 시간 (분)
    • 참여 점수 - 비디오 시청 횟수 1회당 평균 시청 백분율입니다(시청한 백분율을 모두 더한 다음 비디오 시청 건수로 나누어 계산).
    • 일일 순 시청자 -시청자의 IP 주소 및 사용자 에이전트를 사용하여 계산됩니다. 브라이트코브는 브라우저에 의해 쿠키가 차단되고 있기 때문에 타사 쿠키를 사용하지 않기로 결정했습니다. 일일 고유 시청자는 계정 수준에서만 사용할 수 있습니다. 동영상 (또는 다른 항목) 을 기준으로 필터링할 때는 이 지표를 사용할 수 없습니다. 브라이트코브는 또한 고유 시청자의 IP 주소 대신 사용자가 제공하는 가치를 기반으로 할 수 있는 기능을 제공합니다. 자세한 내용은 Analytics 이벤트 추적개발자 문서.
    • 비디오 대역폭 -비디오 전용 대역폭을 포함합니다. 대역폭 데이터는 3-5 일 지연됩니다.
    • 총 대역폭 -비디오, 이미지 및 플레이어로드를위한 대역폭을 포함합니다. 대역폭 데이터는 3-5 일 지연됩니다.

    브라이트코브 Live에 계정이 활성화되어 있는 경우 모든 비디오, VOD 에셋 또는 라이브 에셋에 대한 퍼포먼스 데이터를 볼 수 있는 옵션이 제공됩니다.

    참고:
    • 애널리틱스 보고서에는 계정에서 삭제된 동영상에 대한 분석이 포함됩니다. 삭제된 특정 동영상에 대한 보고서를 보기 위해 애널리틱스 모듈에서 삭제된 비디오를 검색할 수 없습니다. 이러한 신고를 받으려면 삭제 된 동영상의 동영상 ID를 알고 있어야하며분석 API보고서를 생성합니다.
    • 일일 고유 시청자 및 비디오 대역폭 데이터에 대한 데이터를 사용할 수 있기까지 최대 3일이 소요됩니다.
    • 비디오 보기는 참여 보고서의 x% 조회수와 동일하지 않습니다. 비디오 시청수는 비디오가 처음 재생되기 시작할 때 올라갑니다. 1% 시점의 시청 횟수는 첫 1%를 시청했을 때 기록됩니다. 여기서 차이점은 첫 번째 퍼센트가 여러 번 시청되는 경우 (예: 비디오가 자동 재생으로 설정된 경우) 카운트가 증가하지만 비디오가 처음 재생되기 시작하지만 재생시 계산되지 않을 때 비디오 보기는 카운트된다는 것입니다 (사용자가 페이지를 다시 로드하거나재생 목록 등).
    • 동일한 계정 내의 동시 보고서 요청은 연속적으로 실행됩니다. 보고서에 큰 날짜 범위가 있고 많은 필드가 선택된 경우 모든 결과가 반환되기 전에 지연이 있을 수 있습니다. 좋은 대안은 대규모 보고서를 예약하는 것입니다.
    • 와인 페이지 경험 , 동영상이 플레이어에로드 된 것처럼 보이지만 시청자가 재생 버튼 오버레이를 클릭 할 때까지 동영상이로드되지 않습니다. 따라서 시청자가 동영상을 재생할 때까지 Analytics 모듈에서 동영상 노출수를 기록하지 않습니다.

    차트 아래의 보고서 전환기를 사용하여 다른 보고서로 빠르게 전환할 수 있습니다.

    보고서 전환기 오른쪽에 있는 드롭다운 목록을 사용하여 보고서 차원을 선택할 수 있습니다. 성과 보고서에서 사용할 수 있는 차원은 다음과 같습니다.

    이전 기간과 비교

    선택이전 기간과 비교체크 박스는 이전 기간의 데이터를 나타내는 오버레이 점선 그래프를 추가합니다.

    이전 기간은 보고서 날짜 범위에 대해 현재 선택된 기간을 기준으로 합니다. 예를 들어, 최근 7일의 날짜 범위를 선택하면 이전 기간은 2주 전입니다. 주별 추세를 비교할 때는 전체 주를 포함하는 날짜 범위를 선택하는 것이 가장 좋습니다. 예를 들어, 월요일에 시작하는 날짜를 선택하는 경우 종료 날짜를 일요일로 설정합니다. 시작 날짜가 금요일인 경우 종료 날짜를 목요일로 설정합니다. 막대와 파선 위에 마우스를 놓으면 날짜 및 해당 값을 볼 수도 있습니다.

    • 비디오 시청 횟수 - 비디오가 재생되기 시작한 횟수로, 스트리밍이 시작되었을 때 기록됩니다(되감기 또는 다시 재생은 포함되지 않음). 단, 이는 각 시청자를 세는 단위가 아닙니다.
    • 비디오 노출수 - 비디오가 플레이어에 로드되어 상호 작용할 준비가 된 횟수입니다.
    • 재생률 -동영상이로드 된 후 재생 한 사용자 비율 (동영상 조회수를 동영상 노출 수로 나눈 값)

    • 참여 점수 -동영상 조회 당 평균 시청률 (모든 시청률을 동영상 조회수로 나눈 값으로 계산)
