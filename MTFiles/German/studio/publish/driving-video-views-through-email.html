    Steigerung von Videoansichten mittels E-Mail

    In diesem Abschnitt erfahren Sie, wie Sie die Anzahl von Videoansichten mithilfe von E-Mail erhöhen.
     

    Wir werden oft nach der besten Methode zum Einbetten von Video in eine E-Mail gefragt. Auf den meisten Plattformen ist das direkte Einbetten von Videos in E-Mails nicht möglich. Hierbei handelt es sich nicht um eine Einschränkung von Brightcove. Beinahe jedes moderne E-Mail-System entfernt aus Sicherheitsgründen den Code zum Einbetten eines Video-Players. Wir können aber mithilfe eines Teasertextes und eines Miniaturbilds auf ein Video hinweisen und für Zugriffe sorgen. Die einem Videoplayer gleichende Miniaturansicht ist ein anklickbarer Link, der zu unserer Videostartseite führt.

    Für diese Lösung sind die folgenden Schritte auszuführen:

    1. Erstellen Sie einen Brightcove-Player
    2. Erstellen Sie eine Video-Landingpage
    3. Screenshot des Players aufnehmen
    4. E-Mail-Nachricht erstellen

    Erstellen eines Brightcove-Players

    Erstellen Sie zunächst einen dedizierten Brightcove-Player für eine Videostartseite. Mit einem dedizierten Player lassen sich Video-Analytics leichter nachverfolgen. Konfigurieren Sie den Player für die automatische Wiedergabe, damit das Video automatisch gestartet wird, wenn der Benutzer über die E-Mail zur Startseite gelangt.

    1. Melden Sie sich bei Video Cloud Studio an und öffnen Sie das Spieler Modul.
    2. Klicken Sie auf New Player.
    3. Benennen Sie den Player, und klicken Sie auf Save.
    4. Klicken Sie auf den Link für den Player, und öffnen Sie die Player-Eigenschaften.
    5. Klicken Sie im Abschnitt Settings auf Edit.
    6. einstellen Auto-StartVideo beim Laden des Players zu Ja.
    7. Speichern Sie die Änderung, und veröffentlichen Sie den Player.
    8. Klicken Sie auf Embed Code & URL und anschließend auf Published Player.
    9. Wählen Sie den iframe Embed Code, und kopieren Sie diesen.

    Erstellen einer Videostartseite

    Die Videostartseite ist die Seite, auf die der Benutzer über die E-Mail-Nachricht gelangt. Fügen Sie zunächst den Veröffentlichungscode des Iframe-Players hinzu. Die Video-ID, die angezeigt wird, wird als URL-Parameter eingefügt. Hierdurch kann die Startseite zum Anzeigen verschiedener Videos verwendet werden. Die hierzu nötigen Schritte finden Sie weiter unten.

    1. Erstellen Sie für die Videostartseite eine neue HTML-Seite.
    2. Fügen Sie den iframe-Playerveröffentlichungscode ein.
    3. Fügen Sie den JavaScript-Code zum Einlesen des Video-ID-URL-Parameters hinzu. Nachfolgend finden Sie ein Beispiel.

    4. Speichern Sie die Startseite, und veröffentlichen Sie sie. Notieren Sie sich die Seiten-URL.

    Erstellen eines Screenshots vom Player

    Erstellen Sie einen Screenshot vom Player, in dem das Video auf der Startseite abgespielt wird. Dieses Bild wird als anklickbarer Link im E-Mail-Text verwendet. Ein Screenshot kann auf verschiedene Arten erstellt werden:

    • Mac – Apple stellt ein Dokument über das Erstellen eines Screenshots auf dem Mac bereit. Es gibt auch Produkte von Drittanbietern wie Snagit , Voila und andere.
    • Windows – Windows bietet ein Snipping-Tool zum Erstellen von Screenshots. Es gibt auch Produkte von Drittanbietern wie Snagit , Lichtbildschirm und andere.

    Erstellen einer E-Mail-Nachricht

    Jetzt müssen Sie noch eine E-Mail-Nachricht erstellen. Fügen Sie dem E-Mail-Text die Playerabbildung hinzu, und konfigurieren Sie diese als anklickbaren Link, der die Videostartseite öffnet. Die hierzu nötigen Schritte sind vom verwendeten E-Mail-Client abhängig. Die folgenden Schritte gelten für Gmail.

    1. Öffnen Sie Gmail.
    2. Klicken Sie auf Compose.
    3. Klicken Sie in der E-Mail unten auf das Symbol Insert Photo.
    4. Suchen Sie nach der erstellten Playerabbildung. Das Bild wird in die E-Mail eingebettet.
    5. Wählen Sie das gesamte Bild aus, und klicken Sie in der E-Mail unten auf das Symbol Insert link.
    6. Klicken Sie auf den Link Change, um den Abbildungslink zu bearbeiten.
    7. Fügen Sie die URL zur Videostartseite hinzu, und klicken Sie auf OK.
    8. Senden Sie die E-Mail.
