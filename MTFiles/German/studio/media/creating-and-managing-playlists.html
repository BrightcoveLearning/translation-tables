    Erstellen und Verwalten von Wiedergabelisten

    In diesem Abschnitt erfahren Sie, wie Sie Wiedergabelisten mit Video Cloud Studio erstellen und verwalten.
     
     

    Eine Wiedergabeliste enthält mehrere Videos, die in einer bestimmten Reihenfolge gruppiert sind und in einem Player wiedergegeben werden sollen. Das Medienmodul erleichtert das Erstellen von Wiedergabelisten und das Hinzufügen von Videos zu einer Wiedergabeliste. Nach dem Erstellen der Wiedergabeliste können Sie in einem Player eine Wiedergabeliste veröffentlichen. Weitere Informationen zum Veröffentlichen von Wiedergabelisten finden Sie unter Veröffentlichen von Videos und Wiedergabelisten mit dem Medienmodul.

    Weitere Informationen zu Wiedergabelisten, einschließlich Best Practices und Tipps zu deren effektiver Verwendung, finden Sie unter Einführung in Wiedergabelisten.

    Arten von Wiedergabelisten

    Es gibt zwei unterschiedliche Arten von Wiedergabelisten:

    • Manual – bei einer manuellen Wiedergabeliste weisen Sie bestimmte Videos einer Wiedergabeliste zu und sortieren sie manuell.
    • Smart – bei einer intelligenten Wiedergabeliste wird die Liste der Videos in der Wiedergabeliste basierend auf Video-Tags oder anderen Video-Eigenschaften automatisch erstellt. Intelligente Wiedergabelisten sind dynamisch und aktualisieren sich automatisch, je nachdem, welche Videos in der Medienbibliothek enthalten sind und durch welche Regeln die Wiedergabeliste definiert wird.

    Maximal 100 Videos können zu einer Wiedergabeliste hinzugefügt werden (sowohl Manual als auch Smart). Die Anzahl der Wiedergabelisten, die erstellt werden können, ist unbegrenzt. Die Videos, die anfänglich in eine Playlist im Player geladen werden, richten sich nach der Art der Wiedergabeliste:

    • Manuelle Wiedergabelisten - Videos werden basierend auf der Reihenfolge der Videos in der manuellen Wiedergabeliste in die Playlist geladen.
    • Intelligente Wiedergabelisten - Videos werden basierend auf der Playlist-Einstellung der Wiedergabeliste in die Wiedergabeliste geladen.

    Informationen zum programmgesteuerten Abrufen zusätzlicher Videos in eine Wiedergabeliste finden Sie in der Wiedergabe-API-Paging Spielerprobe.

    Erstellen einer Wiedergabeliste

    Öffnen Sie das Medienmodul zum Erstellen einer Wiedergabeliste, und führen Sie folgende Schritte aus:

    1. Klicken Sie im seitlichen Navigationsfenster auf Create Playlist. Das Dialogfeld „New Playlist“ wird geöffnet.
      Hinweis: Blenden Sie das seitliche Navigationsfenster durch einen Klick auf die Schaltfläche des seitlichen Navigationsfensters ein oder aus.
    2. Wählen Sie die Wiedergabelistenart Manual oder Smart aus.
    3. Geben Sie einen Wiedergabelistennamen ein. Beachten Sie dabei die folgenden Richtlinien:
      • Der Name ist auf 100 Zeichen begrenzt
      • Illegaler Charakter:/
    4. (Optional). Geben Sie eine Referenz-ID und eine Beschreibung an.
    5. Aktivieren Sie Show Playlist in Sidebar, damit die Wiedergabeliste in der Seitenleiste erscheint.
      Hinweis: In der Seitenleiste können maximal 100 Wiedergabelisten angezeigt werden.
    6. Klicken Sie auf Save.

    Arbeiten mit intelligenten Wiedergabelisten

    Beim Erstellen einer intelligenten Wiedergabeliste können die folgenden Einstellungen konfiguriert werden:

    • Tags – verwenden Sie Tags zum Definieren der Videos, die in die Wiedergabeliste aufgenommen werden sollen. Das Dropdown „ Tags “ bietet folgende Optionen:
      • enthält eines oder mehrere - Schließen Sie Videos ein, die eines oder mehrere der Tags aufgelistet haben
      • enthält alle - Erfordern, dass alle Videos in der Playlist alle aufgelisteten Tags haben

      Geben Sie im Textfeld "Tags" ein oder mehrere Tags (durch Kommas getrennt) ein, die zum Definieren der Wiedergabeliste verwendet werden.

    • Play Order – legt die Reihenfolge fest, in der die Videos der Wiedergabeliste wiedergegeben werden. Die folgenden Optionen sind verfügbar:
      Spiel-Reihenfolge Beschreibung
      Aktiviertes Datum (Älteste zuerst) Basierend auf dem Datum, an dem das Video aktiviert wurde, zuerst die ältesten Videos
      Aktiviertes Datum (Neueste zuerst) Basierend auf dem Datum, an dem das Video aktiviert wurde, werden zuerst die neuesten Videos
      Videoname (A - Z) Alphabetisch nach Videonamen
      Gesamtzahl der Basierend auf den gesamten Videowiedergaben
      Vergangene Woche spielt Basierend auf der nachfolgenden Woche werden insgesamt Videos abgespielt[1]
      Startdatum (Älteste zuerst) Basierend auf dem geplanten Startdatum des Videos, zuerst die ältesten Videos. Wenn für ein Video kein Startdatum angegeben wurde, wird es in einer nach dem Startdatum sortierten Wiedergabeliste nicht angezeigt. Notiz: Die Videoplanung ist nur für Video Cloud Enterprise-Konten verfügbar.
      Startdatum (neuestes zuerst) Basierend auf dem Datum, an dem das Video beginnen sollte, zuerst die neuesten Videos. Wenn für ein Video kein Startdatum angegeben wurde, wird es in einer nach dem Startdatum sortierten Wiedergabeliste nicht angezeigt. Notiz: Die Videoplanung ist nur für Video Cloud Enterprise-Konten verfügbar.

      Notizen

      • [1] Beachten Sie, dass die in den nachfolgenden Wochen wiedergegebenen Daten, die in den Videometadaten gespeichert sind, bis zu 2 Tage alt sein können. Daher stimmen die Videos, die Sie erhalten, möglicherweise nicht mit denen überein, die Sie sehen, wenn Sie sich die Videoansichten der letzten 7 Tage im Analytics-Modul ansehen.
    • Limit Number of Videos -– legen Sie die maximale Anzahl der Videos für eine Wiedergabeliste fest.

    Hinzufügen von Videos zu einer manuellen Wiedergabeliste

    Um einer Wiedergabeliste Videos hinzuzufügen, gehen Sie wie folgt vor:

    1. Aktivieren Sie die Kästchen neben den Videos, die Sie der Wiedergabeliste hinzufügen möchten.
    2. Klicke oben auf der Seite auf Mehr Zur Playlist > hinzufügen .
    3. Wählen Sie die Playlist aus, zu der Sie die Videos hinzufügen möchten.
    4. Klicken Sie auf Add. Videos, die einer manuellen Wiedergabeliste hinzugefügt wurden, stehen oben in der Wiedergabeliste.

    Ändern der Reihenfolge der Videos in einer Wiedergabeliste

    Je nach der Art der verwendeten Wiedergabeliste sind unterschiedliche Schritte zum Ändern der Video-Reihenfolge notwendig.

    Ändern der Reihenfolge der Videos in einer manuellen Wiedergabeliste

    1. Öffnen Sie die Wiedergabeliste mit einer der folgenden Methoden:
      • Klicken Sie im Seitennavigationsbereich auf den Namen der Playlist
      • Klicken Sie im Seitennavigationsbereich auf Alle Playlists , suchen Sie die Wiedergabeliste und klicken Sie auf den Link zum Namen der Playlist.
    2. Die Liste der Videos in der Wiedergabeliste wird angezeigt.
    3. Klicken Sie dauerhaft auf das Video, und verschieben Sie es an die neue Position in der Reihenfolge. Die dunkle Linie zeigt die neue Position an.
    Hinweis: Sie können die Videoreihenfolge auch ändern, indem Sie das Feld Wiedergabereihenfolge bearbeiten.
    Eine Neuordnung per Drag & Drop erfolgt nicht Arbeiten Sie in IE11 oder Edge, aber Sie können die Elemente neu bestellen, indem Sie die Zahlen in der Spielreihenfolge Feld.

    Ändern der Reihenfolge der Videos in einer intelligenten Wiedergabeliste

    1. Wird die Wiedergabeliste in der Seitenleiste angezeigt, klicken Sie auf die Wiedergabeliste und anschließend auf den Abwärtspfeil neben der Wiedergabeliste. Klicken Sie dann auf Edit Settings.
    2. Wird die Wiedergabeliste nicht in der Seitenleiste angezeigt, klicken Sie oben auf der Navigationsleiste auf All Playlists. Suchen Sie die Wiedergabeliste, und klicken Sie auf den Link des Wiedergabelistennamens. Klicken Sie anschließend auf Edit Settings.
    3. Legen Sie für die Wiedergabeliste eine neue Abspielreihenfolge fest.
    4. Klicken Sie auf Save.

    Intelligente Wiedergabelisten werden dynamisch erstellt, wenn der Player aus den verfügbaren Videoinformationen geladen wird. Daher sollte die Reihenfolge immer den aktuellen Status widerspiegeln. Der Player erhält seine Informationen jedoch immer über die Wiedergabe-API, die Daten verwendet, die bis zu 20 Minuten zwischengespeichert wurden. Das Aktualisieren von Wiedergabelisten kann so lange dauern und hinter dem stehen, was die CMS API würde zeigen.

    Bearbeiten der Wiedergabelisteneinstellungen

    Führen Sie zum Bearbeiten der Einstellungen einer manuellen oder intelligenten Wiedergabeliste folgende Schritte aus:

    1. Wird die Wiedergabeliste in der Seitenleiste angezeigt, klicken Sie auf die Wiedergabeliste und anschließend auf den Abwärtspfeil neben der Wiedergabeliste. Klicken Sie dann auf Edit Settings.
    2. Wird die Wiedergabeliste nicht in der Seitenleiste angezeigt, klicken Sie oben auf der Navigationsleiste auf All Playlists. Suchen Sie die Wiedergabeliste, und klicken Sie auf den Link des Wiedergabelistennamens. Klicken Sie anschließend auf Edit Settings.
    3. Bearbeiten Sie die Einstellungen der Wiedergabeliste, und klicken Sie auf Save.
    Beachte, dass du eine manuelle Wiedergabeliste nicht in eine intelligente Playlist umwandeln darfst oder umgekehrt.

    Wird die Wiedergabeliste in der Seitenleiste angezeigt, klicken Sie auf die Wiedergabeliste, und anschließend auf den Abwärtspfeil neben der Wiedergabeliste. Klicken Sie dann auf Edit Settings.

    Löschen einer Wiedergabeliste

    Führen Sie zum Löschen einer Wiedergabeliste die folgenden Schritte aus:

    1. Klicken Sie zum Anzeigen aller Wiedergabelisten auf All Playlists.
    2. Aktivieren Sie das Kontrollkästchen neben der zu löschenden Wiedergabeliste.
    3. Klicken Sie auf die Schaltfläche Löschen ().
    4. Klicken Sie auf Playlist löschen , um den Löschvorgang zu bestätigen.

    Wird die Wiedergabeliste in der Seitenleiste angezeigt, klicken Sie auf die Wiedergabeliste und anschließend auf den Abwärtspfeil neben der Wiedergabeliste. Klicken Sie dann auf Delete.

    Entfernen von Videos aus einer manuellen Wiedergabeliste

    Führen Sie zum Entfernen eines Videos aus einer manuellen Wiedergabeliste die folgenden Schritte aus:

    1. Öffnen Sie die manuelle Wiedergabeliste mit einer der folgenden Methoden:
      • Klicken Sie im Seitennavigationsbereich auf den Namen der Playlist
      • Klicken Sie im Seitennavigationsbereich auf Alle Playlists , suchen Sie die Wiedergabeliste und klicken Sie auf den Link zum Namen der Playlist.
    2. Die Liste der Videos in der Wiedergabeliste wird angezeigt.
    3. Aktivieren Sie das Kontrollkästchen neben den zu löschenden Videos.
    4. Klicken Sie auf Remove from Playlist.

    Ein Video kann ebenfalls durch das Bearbeiten der Video-Einstellungen aus einer manuellen Wiedergabeliste entfernt werden. In dem Kategorisierung Löschen Sie die Wiedergabeliste aus dem Bereich Manuelle Wiedergabelisten Sektion.
