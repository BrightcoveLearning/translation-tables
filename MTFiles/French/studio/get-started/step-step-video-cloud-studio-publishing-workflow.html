    Étape par étape : Workflow de publication Video Cloud Studio

    Dans cette rubrique, vous allez travailler via un flux de travail de publication vidéo de base à l'aide de Video Cloud Studio. Cette rubrique fournit un ensemble de mesures concrètes que vous pouvez suivre vous-même pour découvrir certaines des fonctionnalités de Video Cloud.

    Ce sujet est un compagnon de la Introduction à Video Cloud cours de formation à la demande. Les exercices pratiques de cette formation sont fondés sur les étapes décrites dans le présent document. Il est recommandé de suivre cette formation, puis d'utiliser ce document afin que vous puissiez effectuer les étapes pratiques par vous-même.

    Conditions préalables

    • Un compte Video Cloud - vous pouvez vous inscrire pour un compte d'essai ici.
    • Vous devriez avoir quelques exemples de vidéos à utiliser ou vous pouvez Télécharger quelques exemples de vidéos
    Remarque : Les étapes et les captures d'écran de cette rubrique utilisent les exemples de vidéos fournis.

    Contour

    Cette rubrique passe par un processus de publication vidéo de base pour une seule vidéo et pour une liste de lecture. Les sujets suivants sont abordés :

    • Téléchargement de vidéos
    • Modification des propriétés vidéo
    • Créer et styliser des joueurs
    • Publication d'une vidéo sur le Web
    • Créer une liste de lecture
    • Publication d'une liste de lecture

    Le module Video Cloud Analytics n'est pas couvert dans le cadre de ce démarrage rapide. Pour en savoir plus sur l'analyse de Video Cloud, regardez le Introduction à Video Cloud Analytics cours de formation à la demande.

    charger des vidéos sur Video Cloud

    Avant de continuer, vous devriez avoir quelques exemples de vidéos que vous pouvez utiliser ou Télécharger les exemples de vidéos.

    Connectez-vous à votre compte Video Cloud

    1. Dans une fenêtre de navigateur, accédez à https://studio.brightcove.com.
    2. Saisissez vos identifiants et cliquez sur Sign In.

    Télécharger des vidéos locales sur Video Cloud

    1. Cliquez sur Télécharger pour ouvrir le module de téléchargement.
    2. Dans le module Upload, sélectionnez un profil d'ingest.
    3. Cliquez sur Rechercher des vidéos et parcourez et sélectionnez vos vidéos ou tous les exemples de vidéos fournis.
    4. La page affichera toutes les vidéos et la progression du téléchargement.

      Tant que les téléchargements sont dans un état de traitement, ne fermez pas ou ne vous éloignez pas de la fenêtre du navigateur. Cela peut entraîner un téléchargement incomplet.

      Lorsque le téléchargement est terminé, un statut Traitement terminé s'affiche. Cela indique que le traitement est terminé et qu'il est sûr de naviguer à partir du module Upload.

      Remarque : Il est possible de cliquer sur le Éditer bouton à côté du nom de fichier pour modifier certaines des métadonnées vidéo. L'ajout de métadonnées vidéo est abordé dans la rubrique Démarrage rapide suivante.

    Télécharger une vidéo à distance vers Video Cloud

    1. dans le Télécharger module, cliquez sur Entrer l'URL.
    2. Entrez l'URL suivante vers un fichier hébergé sur un compte Amazon S3.
      http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4
    3. Cliquez sur Vidéo de processus pour commencer le téléchargement.

    (Facultatif) Télécharger des vidéos à l'aide d'un appareil

    1. À l'aide d'un appareil mobile, ouvrez Video Cloud Studio à l'adresse https://studio.brightcove.com.
    2. Ouvrez le Télécharger module.
    3. Sélectionnez un profil d'ingest.
    4. Cliquez sur Rechercher des vidéos. L'application de rouleau de caméra sur l'appareil devrait s'ouvrir.
    5. Sélectionnez une vidéo à télécharger. Le téléchargement commencera et la progression du téléchargement sera affichée.

    Modification des propriétés de la vidéo

    Dans cette tâche, vous allez modifier les propriétés de la vidéo.

    1. Dans Video Cloud Studio, cliquez sur Media pour ouvrir le module Médias.
    2. Cliquez sur le titre du Pélican brun vidéo, localisez le Information vidéo section puis cliquez sur Éditer. Met le brève description à Pelican on the rocks.
    3. Ajouter un Étiqueter de birds.
    4. Cliquez sur Retour aux vidéos pour revenir à la liste des vidéos.
    5. Cliquez sur le titre du http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4 vidéo, localisez le Information vidéo section puis cliquez sur Éditer. Définissez les propriétés suivantes :
      • Nom: Oystercatcher
      • Brève description: Oystercatcher on the beach
      • Mots clés: birds

      Cliquez sur sauver une fois terminé, puis cliquez sur Retour aux vidéos pour revenir à la liste des vidéos.

    6. Modifiez les Mésange et le Pivert vidéos. Marquer les vidéos avec birds.
    7. (Facultatif) Modifiez certaines des vidéos d'animaux et marquez les vidéos avec animals.

    Création et style d'un lecteur

    Dans cette tâche, vous allez créer et styliser un joueur.

    Création d'un lecteur

    1. Dans Video Cloud Studio, cliquez sur Players pour ouvrir le module Players.
    2. Cliquez sur New Player.
    3. Entrer Custom Player pour le nom et cliquez sur sauver.

    Coiffer le joueur

    1. Clique le Joueur personnalisé lien pour ouvrir les propriétés du lecteur.
    2. Confirmer le Type de joueur est Vidéo unique.
    3. Cliquez sur Style dans le menu de navigation de gauche.
    4. Vérifier Afficher le titre et la description.
    5. Sélectionnez une forme de bouton de lecture et une position du boutonde lecture.
    6. Clique le Principale sélecteur de couleur et réglez la couleur sur une nuance de rouge.
    7. Vérifiez que les changements de couleur sont reflétés dans le lecteur d'aperçu.
    8. Cliquez sur Publier et incorporer> Publier les modifications pour publier les modifications.

    Publication d'une vidéo sur le web

    Dans cette tâche, vous allez publier une vidéo sur le Web.

    Publication d'une seule vidéo

    1. Dans Video Cloud Studio, cliquez sur Media pour ouvrir le module Médias.
    2. Cochez la case en regard de Pélican brun vidéo, puis cliquez sur Publier et incorporer> Publier sur le Web.
      Note : Pour plus d'informations sur les autres options de publication, consultez le document correspondant :
      • Page de destination de l'e-mail - Publication d'une vidéo par e-mail
      • Réseaux sociaux - Publication de vidéos sur des plateformes de médias sociaux
      • Expérience de la galerie sur la page - Publication de vidéos dans une expérience sur la page
    3. Vérifiez que la boîte de dialogue Publication s'ouvre.
    4. dans le Sélectionner un joueur liste déroulante sélectionner Joueur personnalisé. Ce lecteur a été créé dans une section précédente.
      Remarque : Le lecteur par défaut de Brightcove est créé par défaut dans tous les comptes Video Cloud.
    5. Cliquez sur l' URL située sous le lecteur de prévisualisation pour l'ouvrir dans un nouvel onglet de navigateur. Confirmez que la vidéo s'affiche et qu'elle peut être lue.
      Remarque : Cliquez sur Raccourcir pour générer une URL raccourcie.
    6. (Facultatif) Envoyez l'URL à votre appareil mobile. Cliquez sur le lien et confirmez que la vidéo est lue sur votre appareil mobile.

    création de sélections

    Dans cette tâche, vous allez créer une playlist manuelle et intelligente.

    Création d'une liste de lecture manuelle

    1. Dans Video Cloud Studio, cliquez sur Media pour ouvrir le module Médias.
    2. Dans le volet de navigation latéral, cliquez sur le lien Create Playlist. La fenêtre New Playlist s'affiche.
    3. Définissez Playlist type sur Manual.
    4. Définissez Playlist Name sur Favorites.
    5. Cochez l'option Show Playlist in Sidebar pour faire apparaître la sélection dans la barre latérale.
    6. Cliquez sur Save.

    Ajout de vidéos à la liste de lecture manuelle

    1. Cliquez sur All Videos dans le volet de navigation latéral pour afficher toutes les vidéos du compte.
    2. Cochez les cases correspondantes aux vidéos à ajouter à la sélection.
    3. Cliquez sur Plus> Ajouter à la playlist.
    4. Sélectionnez le Favoris playlist, puis cliquez sur Ajouter.
    5. Clique le Favoris playlist dans le panneau de navigation latéral et confirmez que les vidéos ont été ajoutées à la playlist.
    6. Réorganisez l'ordre des vidéos en cliquant sur une ligne et en la faisant glisser sur une nouvelle position dans la liste.

    Création d'une liste de lecture intelligente

    1. Dans le volet de navigation latéral, cliquez sur le lien Create Playlist.
    2. Définissez Playlist type sur Smart.
    3. Définissez Playlist Name sur Birds.
    4. Cochez l'option Show Playlist in Sidebar pour faire apparaître la sélection dans la barre latérale.
    5. Met le Mots clés champ à contient un ou plusieurs et entrez birds pour l'étiquette.
    6. Cliquez sur Save.
    7. Cliquez sur la sélection Birds dans le volet de navigation latéral et vérifiez que les vidéos d'oiseaux ont bien été ajoutées à la sélection.

    Publication d'une sélection

    Dans cet exercice, vous allez créer un lecteur, modifier ses paramètres pour prendre en charge les sélections et publier une sélection.

    Modification des paramètres du lecteur de façon à prendre en charge les sélections

    Remarque : Si vous sélectionnez le Sensible option de dimensionnement, le code Standard (iframe) fonctionne bien et le lecteur et la liste de lecture sont réactifs. Si vous utilisez le code Advanced (intégration dans la page), le joueur sera réactif, mais pas la liste de lecture.
    1. Dans Video Cloud Studio, cliquez sur Players pour ouvrir le module Players.
    2. Cliquez sur New Player. Nommez le lecteur Playlist Player et cliquez sur Save.
    3. Cherchez le lecteur de la sélection et cliquez sur le nom de celui-ci pour ouvrir ses propriétés.
    4. Sélectionner un Type de joueur de Playlist.
    5. Met le Ratio d'aspect à16: 9.
    6. Met le Dimensions du joueur à un Largeur de 640.
    7. Cliquez sur Save.
    8. Cliquez sur Style dans le menu de navigation de gauche.
    9. Clique le Type de liste de lecture de Liste de lecture (verticale).
    10. Cliquez sur Lecture dans le menu de navigation de gauche.
    11. Vérifier Lire la vidéo lorsqu'elle est sélectionnée.
    12. Vérifier Mode de lecture continue.
    13. Cliquez sur Publier et incorporer> Publier les modifications pour publier les modifications sur le lecteur.

    Publication d'une sélection

    1. Cliquez sur Media pour ouvrir le module Médias.
    2. Cliquez sur All Playlists dans le volet de navigation latéral pour afficher toutes les sélections.
    3. Cochez la case correspondante à la sélection Birds.
    4. Cliquez sur Publier et incorporer> Publier sur le Web.
    5. Sélectionnez le lecteur de la sélection.
    6. Cliquez sur le lien URL. Le lecteur se lancera dans une nouvelle fenêtre de navigateur. Vérifiez que le lecteur de liste de lecture s'affiche et que les vidéos de la liste de lecture peuvent être lues.
    7. (Facultatif) Envoyez l'URL du lecteur sur votre dispositif mobile par courriel. Cliquez sur le lien et confirmez que le lecteur de liste de lecture apparaît et que les vidéos de la liste de lecture peuvent être lues.
