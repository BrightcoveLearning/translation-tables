    如何在媒體模組中搜尋影片和播放清單

    在本主題中，您將學習如何使用「媒體」模組搜尋視訊。

    媒體模塊提供了搜索...可用於在“視頻雲”帳戶中搜索視頻的字段。

    您可以執行下列類型的搜尋：

    • 影片搜尋
    • 播放清單搜尋
    • 高級搜索
    • 保存的搜索

    影片搜尋

    術語

    在本節中，術語精確表示搜尋字串以引號括住。例如：

    精確搜尋 非精確搜尋
    「大藍色」 偉大的藍色
    name：“大藍” name：藍色

    請注意，「確切」意味著引號內的所有術語必須匹配，但它們可能不完全匹配。舉例來說，搜尋功"running water"能可能會傳回標題或說明water runs downhill中含有片語的影片。

    若要搜尋您帳戶中的影片，請在 [搜尋...] 中輸入搜尋字詞。欄位。根據預設，搜尋將在視訊上執行。

    若要搜尋多個視訊 ID，請使用下列語法： id: xxx id: xxx id: xxx id: xxx 。例如，id:5985323381001 id:5715142164001 .

    也可以搜索播放清單或輸入進階搜尋字詞。

    將搜尋下列視訊欄位：

    • 影片編號（id）
    • 名稱 （name）
    • 簡短的介紹 （description）
    • 詳細描述 （long_description)）
    • 標籤（tags）
    • 參考編號（reference_id）
    • 自定義字段（custom_fields要么{field_name}）

    視訊欄位的查詢支援不同。有些欄位支援部分字搜尋和詞幹搜尋，有些則不支援。

    搜尋欄位  支援 不支援 範例
    名稱和說明 一個或多個整個單詞的完全匹配; 詞幹支持 沒有部分字支援
    • name:"great blue"將符合"Great Blue Heron"
    • name:"great blu"不符"Great Blue Heron"
    • name:great blu將符合"Great Blue Heron"
    視訊 ID 僅完全相符 不支援部分字詞或詞幹 5591838791001將匹配ID為5591838791001的視頻5591838791將不匹配該視頻的視頻
    標籤 僅符合整個單字 不支援部分字詞或詞幹 tags:publ將不匹配帶有標籤的視頻"public"
    參考 ID 以 Ref ID（前 3 個字符）開頭的部分字符串匹配，整個字符串匹配 沒有部分字符串不以 Ref ID 開頭
    • reference_id:myv匹配參考ID為“ myvid1”的視頻
    • reference_id:"myv"確實不匹配具有參考ID的視頻"myvid1"
    日期欄位   無法使用文字方塊搜尋，可使用進階搜尋  
    個別文字自訂欄位 匹配一個或多個完整單詞，並且發芽支持;對於精確搜索字符串，整個單詞僅匹配 沒有部分字支援
    • custom_fields:place將匹配具有自定義字段值為的視頻"places"
    • custom_fields:plac將不將視頻與自定義字段值匹配"places"
    • custom_fields:"place"將不將視頻與自定義字段值匹配"places"
    個別清單自訂欄位 匹配一個或多個完整單詞，並且發芽支持; 精確搜索字符串，僅匹配整個單詞 沒有部分字支援
    • topic:place將匹配主題列表類型的自定義字段值為“地方”的視頻
    • topic:plac將不將視頻與主題列表類型的自定義字段值匹配"places"
    • topic:"place"將不將視頻與主題列表類型的自定義字段值匹配"places"

    什麼是詞幹？

    支援具有共同搜尋字詞詞詞詞詞詞幹的字詞的視訊欄位。另外，詞幹僅支持輸入整個單詞，而不是部分單詞：

    • 範例 1：搜尋結果running將會傳回包含：running、run、runs
    • 範例 2：搜尋結果不vid會傳回包含以下內容的結果：video

    一般規則

    • 對於完全匹配，請用引號括住字符串
    • 默認情況下，視頻雲假定單詞之間的 OR 邏輯運算符
    • 搜尋不區分大小寫

    範例

    搜尋詞彙 傳回
    financial planning 包含financial OR 的所有影片planning
    "financial planning" 包含financial和的所有影片planning。請注意，引號確保符合的影片會包含finance所有字詞，但可能不是確切的字詞，例如，這裡的搜尋字詞也可能會傳回標題或plan描述
    走路 包含步行、漫步或步行的所有視訊

    播放清單搜尋

    要搜索您帳戶中的播放列表，請在搜索...字段，然後單擊播放清單選項。

    播放清單清單會出現在頁面正文中。

    進階搜尋

    進階搜尋可讓您使用搜尋限定元來搜尋特定欄位中的值。例如，您可以指定只搜尋視訊名稱或標籤欄位。您也可以排除具有特定條款的視訊。例如，您可以搜索包含以下內容的視頻體育但排除包含以下內容的視頻棒球。若要開啟「進階搜尋」對話方塊，請按一下「更多搜尋選項」連結。

    您可以使用下列條件來自訂搜尋：

    • 標籤
    • 日期範圍
    • 要包含的文字
    • 要排除的文字
    • 可玩

    標籤

    進階搜尋可以用來傳回基於視訊標籤的視訊。使用標籤進行搜尋時，有三個選項：

    • 任何 -返回帶有任何標籤的視頻（或）
    • 所有 -返回的視頻將包含所有標籤（與）
    • 沒有 -返回沒有任何這些標籤的視頻

    此示例將返回所有標有或的視頻birds要么animals。

    可以使用刪除/添加按鈕輸入其他標籤標準（）。

    日期範圍

    進階搜尋可根據日期範圍來傳回影片。您可以搜尋下列日期：

    • 創建於created_at）
    • 更新於（updated_at）
    • 排定的開始於（schedule.starts_at）
    • 預定的結束於（schedule.ends_at）

    此範例會傳回在 2016 年 3 月 1 日至 2016 年 3 月 31 日期間建立的所有影片。

    可以使用刪除/添加按鈕輸入其他日期範圍（）。

    要包含的文字

    您可以根據影片中繼資料中的文字，使用進階搜尋來傳回影片。您可以搜尋下列欄位：

    • 所有
    • 名稱
    • 名稱及描述
    • 參考 ID
    • 自訂欄位
    • 特定的自訂欄位

    假設該帳戶有兩個自定義字段，年和學科。這將返回所有帶有年自定義字段等於2016年和學科自定義字段等於鳥類。

    您可以使用刪除/新增按鈕 ( ) 來輸入其他條件。

    要排除的文字

    進階搜尋可以根據影片中繼資料中的文字排除視訊。您可以搜尋下列欄位：

    • 名字
    • 名稱及描述
    • 參考 ID
    • 自訂欄位
    • 特定的自訂欄位

    此示例將不會返回包含以下內容的視頻私人的在名稱或說明字段中。

    您可以使用刪除/新增按鈕 ( ) 來輸入其他條件。

    可播放

    高級搜索可以根據他們的狀態返回視頻：

    • 所有
    • 可播放
    • 不可播放

    輸入搜尋條件後，進階搜尋語法會出現在搜尋方塊中。

    見進階搜尋語法部分以獲取更多信息。

    進階搜尋語法

    當使用「進階搜尋」對話方塊來搜尋視訊時，輸入搜尋條件後，進階搜尋語法就會出現在搜尋方塊中。本節概述搜尋語法。您也可以在搜尋方塊中輸入搜尋語法，而不使用「進階搜尋」對話方塊。若要開啟「進階搜尋」對話方塊，請按一下「更多搜尋選項」連結。

    根據需要標記搜索字詞（返回的視頻必須匹配）或排除（返回的視頻）禁止匹配），添加一個+要么--在學期前立即簽名。

    支援下列搜尋限定元和值。

    適用對象 有效值
    name 字串或以括弧括住的字串
    文字 字符串或帶引號的字符串（name）
    tags 字串或以括弧括住的字串
    reference_id 字串或以括弧括住的字串
    custom_fields -搜索所有自定義字段 字串或以括弧括住的字串
    < custom_field_name >-搜索指定的自定義字段名稱 字串或以括弧括住的字串
    state [1] ACTIVE，INACTIVE，PENDING
    updated_at 日期範圍
    created_at 日期範圍
    schedule.starts_at 日期範圍
    schedule.ends_at 日期範圍
    published_at [2] 日期範圍
    complete [3] true或false

    注意事項

    [1]當視頻已共享到一個帳戶但未被接受時，該視頻將處於PENDING狀態。

    [2]視頻的上次激活日期。

    [3]當視頻具有至少一個可播放的再現形式時，該視頻具有完全為true的狀態。這並不一定意味著所有轉碼都已完成。

    使用日期字段

    使用日期欄位時，日期可以是 UTC (2012-08-01T00：00:00 Z) 格式或使用相對格式：

    • -1day
    • -2days
    • +4hours
    • -60minutes
    • -1d
    • -2d
    • +4h
    • -60m

    範例

    搜尋詞彙 傳回
    +name:florida 所有帶有的視頻Florida在影片名稱中
    +text:meeting 所有帶有單詞的視頻meeting在影片名稱或簡短描述中
    +tags:birds 所有帶標籤的視頻birds
    +tags:birds +state:inactive 所有帶標籤的視頻birds並且狀態為inactive
    +tags:live,"now playing" +state:ACTIVE 所有帶標籤的視頻live和now playing並且狀態為active
    +custom_fields:"2015" 所有價值為的視頻2015在任何自定義字段
    +subject:birds 所有價值為的視頻birds在裡面subject自定義字段
    +producer:"Bob Bailey" 所有價值為的視頻Bob Bailey在裡面producer自定義字段
    +created_at:-3d 過去 3 天內建立的所有影片
    +updated_at:-1d 過去一天更新的所有影片

    儲存搜尋

    您可以儲存您經常執行的搜尋，因此可以快速重新執行，而無需重新輸入搜尋語法。輸入返回您要觀看的視頻的搜索語法後，請單擊保存搜索。輸入保存的搜索的名稱，然後單擊保存。

    要重新運行保存的搜索，請點擊保存的搜索搜索框下的鏈接，然後單擊保存的搜索以運行。

    要刪除任何已保存的搜索，請點擊刪除圖標（）旁邊的搜索。

    忽略的單字

    某些字在搜尋字串中會被忽略，因為它們非常常普遍，因此可能會傳回許多與您實際搜尋內容無關的結果。以下是搜尋所忽略的單字清單：

    是 如 在
    是 但 通過

    上 或 這樣
