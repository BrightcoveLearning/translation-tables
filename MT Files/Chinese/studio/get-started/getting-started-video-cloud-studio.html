    Video Cloud Studio入門

    本主題提供視訊雲端 Studio 介面和模組的概觀。視頻雲工作室允許您管理跨桌面，平板電腦和移動設備的視頻發布工作流程。
     

    要訪問Video Cloud Studio，請導航至： https://studio.brightcove.com。使用您的電子郵件地址和密碼登錄。主頁提供以下功能：

    • 查看系統通知
    • 管理視頻雲設置
    • 選擇一個帳戶並查看用戶個人資料
    • 訪問視頻雲模塊
    • 查看視頻雲儀表板
    注意：視頻雲工作室會話超時為 24 小時。

    查看系統通知

    Video Cloud Studio標頭將顯示一個響鈴圖標有可用的系統通知時。按一下圖示將顯示系統通知清單。

    Video Cloud Studio標頭將顯示警告圖標有系統問題時。點擊圖示將開啟布萊特灣系統狀態頁面。

    管理設定

    可從以下位置訪問Video Cloud Studio設置菜單管理員菜單。視訊雲端管理功能表可讓您存取：

    • 管理用戶 -管理有權使用Video Cloud Studio的用戶
    • 帳戶信息 -查看有關當前視頻雲帳戶的信息
    • 上載設定 -提供自定義上傳模塊的選項
    • 攝取個人資料 -管理您的Video Cloud帳戶中的提取配置文件
    • 視頻場 -根據需要標記標準視頻字段，還可以查看，創建和修改帳戶的自定義元數據字段
    • 媒體分享 -將視頻分享到其他帳戶
    • 播放器設定 -提供使用基本用戶名和密碼身份驗證訪問播放器管理和傳送系統API的選項
    • 實時設置 -管理實時模塊使用的設置
    • API認證 -創建和管理API憑據
    • 帳單 -檢視帳戶的帳單資訊
    注意：您必須是帳戶管理員才能存取管理員功能表。

    選取帳戶

    如果您可以存取多個視訊雲端帳戶，視訊雲端帳戶選擇器可用來選取您要存取的視訊雲端帳戶。要切換帳戶，請點擊您的名稱-帳戶在Video Cloud Studio的右上角。當前帳戶在帳戶名稱之前用點號表示。

    按一下「我的設定檔」連結以更新您的使用者資訊和密碼。有關更多信息，請參見管理個人資料信息。

    存取模組

    視頻雲工作室菜單提供了訪問工作室模塊。通過單擊進入菜單家。

    請注意，顯示的模組取決於您的 Video Cloud 訂閱和使用者設定檔。

    • 上載 -將視頻上傳到您的視頻雲帳戶
    • 媒體 -設置視頻屬性，使用文件夾組織視頻並發布視頻
    • 分析工具 -查看有關視頻和播放器性能的分析
    • 玩家們 -創建播放器並設置樣式，並為播放器生成發布代碼
    • 生活 -創建和直播現場直播
    • 雲播出 -為視頻創建實時線性頻道
    • 畫廊 -為您的視頻內容創建視頻門戶
    • 社會的 -將視頻分發到社交媒體帳戶
    • 聽眾 -將視頻分析從Video Cloud連接到營銷自動化平台

    檢視儀表板

    視頻雲工作室首頁顯示視頻雲儀表板。儀表板包含一系列「資訊卡」，可針對您帳戶中的影片提供高水準指標。以下是可用的卡片：

    • 影片觀看次數 -顯示所選時間段內每天的視頻觀看次數。點擊查看視頻查看詳細信息鏈接將打開效果報告，您可以在其中查看視頻效果。
    • 帳戶匯總 -顯示帳戶中的視頻和播放器總數，以及最近創建和更新的視頻數。按一下數字即可開啟媒體或播放器模組。
    • 設備類型 -顯示所選時間段內用於觀看視頻的設備的類型。點擊按設備類型查看視頻觀看次數鏈接將打開效果報告，您可以在其中按設備類型查看視頻性能。
    • 什麼是新的？ -顯示Video Cloud Enterpriseduct更新信息。有關更新的完整列表，請參見Video Cloud Studio發行說明。
    • 熱門視頻 -顯示所選時間段內觀看次數最多的視頻。按一下卡片標題連結，即可開啟「效能報告」，您可以在此檢視影片表現。
    • 查看區域 -顯示選定時間段內的觀看者位置。點擊查看按國家/地區觀看的視頻將打開效果報告，您可以在其中按國家/地區查看視頻觀看次數。

    本地化設定

    視頻雲工作室語言可以通過更改您的個人資料中的語言來更改。支持以下語言：

    • 英文
    • 西班牙文 (西班牙)
    • 日文
    • 法文 (法國)
    • 繁體中文
    • 韓文

    有關更多信息，請參見管理個人資料信息。
