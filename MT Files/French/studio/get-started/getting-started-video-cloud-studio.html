    Premiers pas avec Video Cloud Studio

    Cette rubrique fournit une vue d'ensemble de l'interface et des modules Video Cloud Studio. Video Cloud Studio vous permet de gérer votre flux de travail de publication vidéo sur les ordinateurs de bureau, tablettes et appareils mobiles.
     

    Pour accéder à Video Cloud Studio, accédez à: https://studio.brightcove.com. Connectez-vous en indiquant votre adresse électronique et votre mot de passe. La page d'accueil offre la possibilité de :

    • Afficher les notifications système
    • Gérer les paramètres de Video Cloud
    • Sélectionnez un compte et affichez le profil utilisateur
    • Accéder aux modules Video Cloud
    • Afficher le tableau de bord de Video Cloud
    Remarque : Le délai d'expiration de la session Video Cloud Studio est de 24 heures.

    Affichage des notifications système

    L'en-tête de Video Cloud Studio affichera une icône de cloche lorsque des notifications système sont disponibles. Un clic sur cette icône permet d'afficher la liste des notifications système.

    L'en-tête de Video Cloud Studio affichera une icône d'avertissement lorsqu'il y a des problèmes de système. Cliquez sur l'icône pour ouvrir la page État du système Brightcove.

    Gestion des paramètres

    Le menu des paramètres de Video Cloud Studio est accessible depuis le ADMIN menu. Le menu Admin de Video Cloud permet d'accéder à :

    • gérer les utilisateurs - Gérer les utilisateurs autorisés à utiliser Video Cloud Studio
    • Information sur le compte - Afficher des informations sur le compte Video Cloud actuel
    • Télécharger les paramètres - Fournit des options pour personnaliser le module de téléchargement
    • Profils d'ingestion - Gérez les profils d'ingestion dans votre compte Video Cloud
    • Champs vidéo - Marquez les champs vidéo standard comme requis et affichez, créez et modifiez également les champs de métadonnées personnalisés pour le compte
    • Partage de médias - Partager des vidéos sur d'autres comptes
    • Paramètres du joueur - Offre la possibilité d'utiliser l'authentification de base par nom d'utilisateur et mot de passe pour accéder aux API du système de gestion des joueurs et de distribution
    • Paramètres en direct - Gérer les paramètres utilisés par le module Live
    • Authentification API - Créer et gérer les informations d'identification de l'API
    • Facturation - Afficher les informations de facturation pour le compte
    Remarque : vous devez être un gestionnaire de compte pour accéder au menu ADMIN.

    Sélection d'un compte

    Si vous avez accès à plusieurs comptes Video Cloud, la liste déroulante de sélection de compte Video Cloud permet de sélectionner le compte Video Cloud auquel vous souhaitez accéder. Pour changer de compte, cliquez sur votre nom - compte dans le coin supérieur droit de Video Cloud Studio. Le compte actuel est indiqué par un point placé en regard du nom de compte.

    Cliquez sur le lien Mon profil pour mettre à jour vos informations d'utilisateur et votre mot de passe. Pour plus d'informations, reportez-vous à la section Gestion des informations de profil.

    Accès aux modules

    Le menu Video Cloud Studio permet d'accéder aux modules Studio. Accédez au menu en cliquant DOMICILE.

    Notez que les modules qui s'affichent sont basés sur votre abonnement Video Cloud et votre profil utilisateur.

    • Télécharger - Téléchargez des vidéos sur votre compte Video Cloud
    • Médias - Définissez les propriétés vidéo, organisez les vidéos à l'aide de dossiers et publiez des vidéos
    • Analytique - Affichez des analyses sur les performances de la vidéo et du lecteur
    • Joueurs - Créer et styliser les joueurs et générer du code de publication pour les joueurs
    • Vivre - Créer et diffuser des événements en direct
    • Lecture dans le cloud - Créez des chaînes linéaires en direct pour les vidéos
    • Galerie - Créez des portails vidéo pour votre contenu vidéo
    • Social - Distribuez des vidéos sur les comptes de réseaux sociaux
    • Public - Connecte l'analyse vidéo de Video Cloud aux plateformes d'automatisation du marketing

    Affichage du tableau de bord

    La page d'accueil de Video Cloud Studio affiche le tableau de bord Video Cloud. Le tableau de bord contient une série de « cartes » qui fournissent des statistiques de haut niveau sur les vidéos de votre compte. Les cartes suivantes sont disponibles :

    • Vues vidéo - Affiche le nombre de vues vidéo par jour pour la période sélectionnée. En cliquant sur le Voir les détails de la vidéo Le lien ouvrira le rapport sur les performances où vous pourrez voir les performances vidéo
    • Relevé de compte - Affiche le nombre total de vidéos et de lecteurs dans le compte et le nombre de vidéos récemment créées et mises à jour. Cliquez sur un numéro pour ouvrir le module Media ou Lecteurs.
    • Type d'appareil - Affiche le type d'appareils utilisés pour regarder des vidéos pendant la période sélectionnée. En cliquant sur le Voir les vues vidéo par type d'appareil Le lien ouvrira le rapport sur les performances où vous pourrez afficher les performances vidéo par type d'appareil.
    • Quoi de neuf? - Affiche les informations de mise à jour de Video Cloud Enterpriseduct. Pour une liste complète des mises à jour, consultez le Notes de mise à jour de Video Cloud Studio.
    • Vidéos les plus populaires - Affiche les vidéos avec le plus de vues pour la période sélectionnée. Cliquez sur le lien du titre de la carte pour ouvrir le rapport de performance dans lequel vous pouvez visualiser les performances vidéo.
    • Affichage des régions - Affiche les emplacements du spectateur pour la période sélectionnée. En cliquant Voir les visionnages vidéo par pays ouvrira le rapport sur les performances dans lequel vous pourrez voir les vues vidéo par pays.

    Localisation

    La langue de Video Cloud Studio peut être modifiée en changeant la langue de votre profil. Les langues suivantes sont prises en charge :

    • Anglais
    • Espagnol (Espagne)
    • Japonais
    • Français (France)
    • Chinois (traditionnel)
    • Coréen

    Pour plus d'informations, reportez-vous à la section Gestion des informations de profil.
