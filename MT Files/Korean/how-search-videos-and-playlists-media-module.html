    미디어 모듈에서 비디오 및 재생 목록을 검색하는 방법

    이 항목에서는 미디어 모듈을 사용하여 비디오를 검색하는 방법을 배우게됩니다.

    미디어 모듈은검색... Video Cloud 계정에서 비디오를 검색하는 데 사용할 수있는 필드입니다.

    다음과 같은 유형의 검색을 수행할 수 있습니다.

    • 비디오 검색
    • 재생 목록 검색
    • 고급 검색
    • 저장된 검색

    동영상 검색

    분류: 용어

    이 섹션에서 용어정확한검색 문자열이 따옴표로 묶여 있음을 의미합니다. 예:

    정확한 검색 정확하지 않은 검색
    “그레이트 블루” 그레이트 블루
    name : "그레이트 블루" name : 그레이트 블루

    “정확한”은 따옴표 안의 모든 용어가 일치해야 하지만 정확히 일치하지 않을 수 있음을 의미합니다. 예를 들어"running water"다음 문구가 포함 된 동영상을 반환 할 수 있습니다. water runs downhill제목이나 설명에서.

    계정에서 비디오를 검색하려면검색...들. 기본적으로 검색은 비디오에서 수행됩니다.

    여러 동영상 ID를 검색하려면 다음 구문을 사용하세요. id : xxx id : xxx id : xxx . 예를 들면id:5985323381001 id:5715142164001 .

    검색도 가능합니다재생 목록또는 입력고급 검색어 .

    다음 비디오 필드가 검색됩니다.

    • 동영상 ID ( id )
    • 이름 ( name )
    • 간단한 설명 ( description )
    • 긴 설명 ( long_description) )
    • 태그 ( tags )
    • 참조 ID ( reference_id )
    • 사용자 정의 필드 ( custom_fields또는{field_name} )

    비디오 필드에 대한 쿼리 지원은 다릅니다. 일부 필드는 부분 단어 검색을 지원하고형태소 검색 , 일부는 그렇지 않습니다.

    검색 필드  지원 지원하지 않음 예
    이름 및 설명 정확한하나 이상의 전체 단어와 일치합니다. 형태소 분석지원하다 부분 단어 지원 안 함
    • name:"great blue"일치할 것입니다. "Great Blue Heron"
    • name:"great blu"일치하지않습니다 . "Great Blue Heron"
    • name:great blu일치할 것입니다. "Great Blue Heron"
    비디오 ID 정확한 일치만 일부 단어 없음 또는형태소 분석지원하다 5591838791001 ID가 5591838791001 인 동영상과 일치합니다. 5591838791의지아니그 비디오와 일치
    태그 전체 단어에 대해서만 일치 일부 단어 없음 또는형태소 분석지원하다 tags:publ태그가있는 비디오가 일치하지않습니다 . "public"
    참조 ID 참조 ID (처음 3 자) 로 시작하고 전체 문자열과 일치하는 부분 문자열 일치 참조 ID로 시작하지 않는 부분 문자열 없음
    • reference_id:myv참조 ID가 "myvid1"인 비디오와 일치
    • reference_id:"myv"않습니다아니참조 ID와 동영상 일치"myvid1"
    날짜 필드   텍스트 상자를 사용하여 검색할 수 없음, 고급 검색을 사용하여 검색 가능  
    개별 텍스트 사용자 정의 필드 하나 이상의 전체 단어와 일치하고형태소 분석지원하다; ...에 대한정확한검색 문자열, 전체 단어 일치 만 부분 단어 지원 안 함
    • custom_fields:place사용자 정의 필드 값이있는 비디오를"places"
    • custom_fields:plac의지아니사용자 정의 필드 값이있는 비디오 일치"places"
    • custom_fields:"place"의지아니사용자 정의 필드 값이있는 비디오 일치"places"
    개별 목록 사용자 정의 필드 하나 이상의 전체 단어와 일치하고형태소 분석지원하다; 정확한검색 스팅, 전체 단어 일치 만 부분 단어 지원 안 함
    • topic:place "places"의 주제 목록 유형 사용자 정의 필드 값과 비디오를 일치시킵니다.
    • topic:plac의지아니주제 목록 유형 사용자 정의 필드 값과 비디오 일치"places"
    • topic:"place"의지아니주제 목록 유형 사용자 정의 필드 값과 비디오 일치"places"

    형태소 분석이란 무엇입니까?

    형태소 분석이 지원되는 비디오 필드는 검색 단어의 줄기가 공통인 단어를 반환합니다. 또한 형태소 분석은 부분 단어가 아닌 전체 단어 만 입력하는 것을 지원합니다.

    • 예 1: 검색 중running다음을 포함하는 결과를 반환합니다. running , run , runs
    • 예 2: 검색 중vid다음을 포함하는 결과를 반환하지 않습니다. video

    일반 규칙

    • 정확히 일치하려면 문자열을 따옴표로 묶으십시오.
    • 기본적으로 Video Cloud는 단어 사이의 OR 논리 연산자로 가정합니다.
    • 대소문자를 구분하지 않는 검색

    예

    검색 용어 반품
    financial planning financial OR 포함 된 모든 비디오planning

    고급 검색

    고급 검색을 사용하면 검색 한정자를 사용하여 특정 필드의 값을 검색할 수 있습니다. 예를 들어 비디오 이름 또는 태그 필드만 검색하도록 지정할 수 있습니다. 특정 용어가 포함된 동영상을 제외할 수도 있습니다. 예를 들어 다음이 포함 된 동영상을 검색 할 수 있습니다. 스포츠하지만 다음이 포함 된 동영상은 제외야구 . 고급 검색 대화 상자를 열려면더 많은 검색 옵션링크.

    다음 기준을 사용하여 검색을 사용자 정의할 수 있습니다.

    • 태그

    태그

    고급 검색을 사용하여 비디오 태그를 기반으로 비디오를 반환할 수 있습니다. 태그를 사용하여 검색할 때 다음과 같은 세 가지 옵션이 있습니다.

    • 어떤 -태그 (OR)가있는 동영상 반환
    • 모두 -반환 동영상은 모든 태그 (AND)
    • 없음 -이러한 태그가없는 동영상 반환

    이 예는 다음 중 하나로 태그가 지정된 모든 동영상을 반환합니다. birds또는animals .

    삭제 / 추가 버튼 ()을 사용하여 추가 태그 기준을 입력 할 수 있습니다. ).

    날짜 범위

    고급 검색을 사용하여 날짜 범위에 따라 동영상을 반환할 수 있습니다. 다음 날짜를 검색할 수 있습니다.

    • 만든 날짜 ( created_at )
    • 업데이트 날짜 ( updated_at )
    • 예정된 시작 시간 ( schedule.starts_at )
    • 예정된 종료 시간 ( schedule.ends_at )

    이 예에서는 2016년 3월 1일부터 2016년 3월 31일 사이에 만들어진 모든 동영상을 반환합니다.

    추가 날짜 범위는 삭제 / 추가 버튼 ( ).

    고급 검색 구문

    고급 검색 대화 상자를 사용하여 비디오를 검색하는 경우 검색 조건을 입력한 후 검색 상자에 고급 검색 구문이 나타납니다. 이 섹션에서는 검색 구문에 대해 간략하게 설명합니다. 고급 검색 대화 상자를 사용하지 않고도 검색 구문을 검색 상자에 입력할 수도 있습니다. 고급 검색 대화 상자를 열려면더 많은 검색 옵션링크.

    검색어를 필수로 표시하려면 (반환 된 동영상절대로 필요한 것일치) 또는 제외 (반환 된 동영상안돼일치), 추가 +또는 -용어 바로 앞에 서명하십시오.

    다음 검색 한정자와 값이 지원됩니다.

    한정자 유효한 값
    name 스트링 또는 인용된 스트링
    텍스트 문자열 또는 인용 된 문자열 ( name )
    tags 스트링 또는 인용된 스트링
    reference_id 스트링 또는 인용된 스트링
    custom_fields -모든 사용자 정의 필드 검색 스트링 또는 인용된 스트링
    < custom_field_name >-지정된 사용자 정의 필드 이름을 검색합니다. 스트링 또는 인용된 스트링
    state [1] ACTIVE , INACTIVE , PENDING
    updated_at 날짜 범위
    created_at 날짜 범위
    schedule.starts_at 날짜 범위
    schedule.ends_at 날짜 범위
    published_at [2] 날짜 범위
    complete [삼] true또는false

    참고

    [1]동영상이 계정에 공유되었지만 수락되지 않은 경우 PENDING 상태가됩니다.

    [2]비디오가 마지막으로 활성화 된 날짜입니다.

    [삼]동영상에 재생 가능한 변환이 하나 이상있는 경우 동영상은 완전한 상태가 true입니다. 반드시 모든 트랜스코딩이 완료되었음을 의미하지는 않습니다.

    검색 저장

    자주 실행하는 검색을 저장하여 검색 구문을 다시 입력하지 않고도 신속하게 다시 실행할 수 있습니다. 보고 싶은 비디오를 반환하는 검색 구문을 입력 한 후검색 저장 . 저장된 검색의 이름을 입력 한 다음저장 .

    저장된 검색을 삭제하려면 삭제 아이콘 ( ) 검색 옆에 있습니다.
