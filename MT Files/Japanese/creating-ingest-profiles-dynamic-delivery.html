    動的配信用の取り込みプロファイルの作成

    このトピックでは、動的配信トランスコーディングで使用するカスタムインジェストプロファイルの作成方法の概要について説明します。
     

    取り込みプロファイルには、動画のトランスコード時に使用されるトランスコード設定が含まれています。表示される取り込みプロファイルエディターは、アカウントに設定されているトランスコーディングオプションに基づいて変更されます。

    注：取り込みプロファイルを作成および編集できる機能は、エンタープライズアカウントとプロフェッショナルアカウントを持つパブリッシャーのみが利用できます。

    [プロファイルの取り込み]管理ページにアクセスするには、[ 管理者リンクしてからプロファイルの取り込み。取り込みプロファイルのリストが表示されます。

    注：[プロファイルの取り込み] ページにアクセスするには、管理者ロールを持つ Video Cloud Studio ユーザである必要があります。

    Video Cloud アカウントには、1 つまたは複数のインジェストプロファイルが設定されます。すべての Video Cloud アカウントは、編集または削除できない初期設定の取り込みプロファイルで設定されています。これらはでマークされていますタイプの標準VODスタジオで。[ Profile Version ] には、プロファイルで使用されるトランスコーディングのタイプが表示されます。Brightcoveが提供する取り込みプロファイルで提供されるレンディションの詳細については、を参照してください。動的配信とコンテキストアウェアエンコーディングの標準的な取り込みプロファイル。

    新しい動的配信プロファイルの作成

    [Ingest Profiles] 管理ページを使用して、新しい取り込みプロファイルを作成できます。取り込みプロファイルは JSON を使用して定義されます。

    新しい動的配信インジェストプロファイルを作成する手順は、次のとおりです。

    1. Studioヘッダーの[ADMIN]リンクをクリックします。
    2. [トランスコードプロファイル]リンクをクリックします。
    3. [ 新規プロファイル] [> 動的配信 ] をクリックします。プロファイルエディタが開きます。
    4. プロファイルの [ 名前 ] と [ 説明 ] を入力します。
    5. マスターファイルを保存するかどうかを決定します。マスターを保存すると、マスターファイルのコピーが保持され、後でビデオを再変換できます。また、再度アップロードする必要はありません。マスターのアーカイブの詳細については、マスターのアーカイブに関するドキュメントを参照してください。
    6. 入力します幅そして高さのためにポスターそしてサムネイル画像。
    7. 画像の設定を選択します。
      • アップロード時に画像を自動的にキャプチャ-動画がアップロードされると、Video Cloud はポスター画像とサムネイル画像をキャプチャします。画像は、動画の 50% ポイントでキャプチャされます。
      • 画像をキャプチャしないでください。自分の画像をアップロードまたはキャプチャします -ポスターとサムネイル画像はキャプチャされたまたはアップロードメディアモジュールを使用します。
    8. プロファイルに必要なレンディションを追加します。動画が音声付きで再生されるように、必ず動画レンディションと音声レンディションを両方とも選択してください。ビデオでダウンロード可能なレンディションを利用できるようにする場合は、プログレッシブ (MP4) レンディションも追加します。[ レンディションを追加 ] をクリックします。
      注
      • 使用可能なオーディオとレンディションを表示するには、[ レンディションリストの表示 ] リンクをクリックします。
      • 1080p レンディションをダウンロードできるようにするには、カスタム取り込みプロファイルを作成し、次のレンディションの少なくとも 1 つが含まれていることを確認する必要があります。
        • progressive3500
        • progressive4000
      • では、プログレッシブレンディションのみを生成するプロファイルを作成できますが、そのプロファイルを使用してビデオをアップロードすると、変換中にサムネイル画像とポスター画像はキャプチャされません。
    9. [ レンディションを追加 ] をクリックして、追加のレンディションを追加します。
    10. 使用コーデック、およびレンディションドロップダウンリストを使用して、含めるオーディオ、ビデオ（H264、HEVC）、またはプログレッシブ（MP4）レンディションを選択します。レンディションの詳細がテーブルに表示されます。
    11. [ 保存 ] をクリックして、新しいプロファイルを保存します。

    ソーシャルクリッピング用のカスタムプロファイルの作成

    カスタム取り込みプロファイルは、ソーシャルクリッピングにも使用できます。一般的に、Brightcove のデフォルトインジェストプロファイルに含まれる低品質のレンディションは、Social ディストリビューションのユースケースでは必要ありません。したがって、カスタムクリッピングインジェストプロファイルは、通常、中～高品質のレンディションの数を少数に制限することができます。たとえば、video3500、video2000、および video1700 ビデオレンディションを含むカスタムクリッピング取り込みプロファイルは、ほとんどの Social ディストリビューションの使用例では十分ではありません。ソーシャルクリッピングにカスタムプロファイルを使用すると、透かし画像の追加など、クリップに追加の変換を適用できます。透かし付きのカスタムプロファイルを作成するには、ブライトコーブサポートにお問い合わせください。

    取り込みプロファイルの複製

    既存のプロファイルに類似したプロファイルを作成する必要がある場合は、既存のプロファイルを複製し、編集して必要な変更を加えることができます。

    取り込みプロファイルを複製するには、次の手順に従います。

    1. Studioヘッダーの[ADMIN]リンクをクリックします。
    2. [トランスコードプロファイル]リンクをクリックします。
    3. 複製する取り込みプロファイルをクリックします。
    4. [ 複製 ] をクリックします。
    5. プロファイルエディタを使用して、プロファイル名と定義を編集します。レンディションを削除するには、レンディションの隣にある削除アイコン ( ) をクリックします。
    6. [保存]をクリックします。

    プロファイルでの DRM サポート

    動画の DRM パッケージングのデフォルトを設定するには、ブライトコーブサポートにお問い合わせください。取り込みプロファイルで DRM パッケージを指定する必要はありません。アカウントで DRM が有効になっていて、一部のタイトルを DRM なしで配信する場合は、Media モジュールでビデオのプロパティを編集し、ビデオの DRM フラグを [ DRM なし ] に設定します。

    注：DRM 対応 Brightcove Player の作成の詳細については、「 DRM プラグイン」トピックを参照してください。

    トランスコードプロファイルを編集する

    トランスコードプロファイル管理ページを使用して、トランスコードプロファイルを編集できます。初期設定の取り込みプロファイル（標準VODとマークされている）は編集できないことに注意してください。

    トランスコードプロファイルを編集するには、次の手順を行います。

    1. Studioヘッダーの[ADMIN]リンクをクリックします。
    2. [トランスコードプロファイル]リンクをクリックします。
    3. 編集するトランスコードプロファイルをクリックします。
    4. プロファイルエディターを使用して、プロファイル定義を編集します。レンディションを削除するには、レンディションの隣にある削除アイコン ( ) をクリックします。
    5. [保存]をクリックします。
    注：基本エディターを使用してプロファイルに標準レンディションを追加できますが、カスタムレンディションを使用して追加する必要があります高度なエディター。

    トランスコードプロファイルを削除する

    トランスコードプロファイル管理ページを使用して、必要なくなったトランスコードプロファイルを削除できます。デフォルトの取り込みプロファイル（ Standard VOD とマークされている）は削除できませんが、必要でない場合は非表示にすることができます。

    トランスコードプロファイルを削除するには、次の手順を行います。

    1. Studioヘッダーの[ADMIN]リンクをクリックします。
    2. [トランスコードプロファイル]リンクをクリックします。[Ingest Profiles] 管理ページが開き、アカウント用に作成されたインジェストプロファイルが表示されます。
    3. 取り込みプロファイルの横にある削除アイコン（）をクリックするか、プロファイル名をクリックし、[ Delete ] ボタンをクリックします。
    4. [プロファイルの削除]をクリックして、削除を確認します。

    デフォルトの取り込みプロファイルの選択

    動画のアップロード時とSocial モジュールを使用したクリップの作成時に、デフォルトの取り込みプロファイルを設定できます。アップロードのデフォルトプロファイルは、プロファイル名の横に表示されます。ソーシャルクリッピングのデフォルトプロファイルは、プロファイル名の横に表示されます。

    アップロード/クリッピング用のデフォルトのインジェストプロファイルを選択するには：

    1. Studioヘッダーの[ADMIN]リンクをクリックします。
    2. [トランスコードプロファイル]リンクをクリックします。[Ingest Profiles] 管理ページが開き、アカウント用に作成された取り込みプロファイルが表示されます。
    3. デフォルトとして設定するプロファイルの取り込みプロファイル名をクリックします。
    4. ページ上部の「アップロード」または「ソーシャル」クリッピングリンクをクリックして、現在のプロファイルをデフォルトに設定します。
    注：動的配信プロファイルのみがソーシャルクリッピングのデフォルトプロファイルに設定できます。CAEプロファイルはサポートされていません。

    取り込みプロファイルの非表示と表示

    取り込みプロファイルを非表示にするには、プロファイルに関連付けられたスライダをクリックして [ 非表示] の位置にします。これは、必要のないデフォルトのプロファイルを非表示にする場合に便利です。

    非表示になっている取り込みプロファイルは、アップロードモジュールの [Ingest Profile] リストに表示されません。

    アドバンスエディターの使用

    取り込みプロファイルは JSON を使用して定義されます。プロファイルの JSON 構文を表示するには、プロファイルを編集し、[ 詳細エディター ] タブをクリックします。プロファイルの JSON 表現が表示されます。

    JSON はここで編集できますが、ブライトコーブでは Simple Editor を使用してプロファイルを編集し、間違いの可能性を減らすことを推奨しています。使用可能な属性の詳細については、「動的配信およびコンテキスト認識エンコーディングの取り込みプロファイル API フィールドリファレンス」を参照してください。
