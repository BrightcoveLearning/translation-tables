    Mediaモジュールを使用して動画のプロパティ値を編集する

    このトピックでは、Mediaモジュールで動画プロパティを編集する方法を学習します。

    動画のプロパティを表示、編集するには、Mediaモジュールを開き、動画名をクリックします。動画プロパティページが開きます。

    注：複数の動画の動画プロパティを同時に編集するには、クイック編集（英語ページ）機能を使用します。

    動画プロパティページ上部のボタンをクリックすると、以下ができます。

    • 新しいポスターとサムネイル画像をキャプチャする
    • 動画をフォルダーに移動する、または動画をプレイリストに追加する
    • 動画をアクティブ化/非アクティブ化する
    • ビデオを公開する

    動画がアクティブな場合、[非アクティブ化]ボタンを使用して、動画を非アクティブ化できます。動画が非アクティブな場合、[アクティブ化]ボタンを使用して、動画をアクティブ化できます。非アクティブな動画はBrightcove Playerで再生できません。

    ページ上部のプレビュープレーヤーを使用して、動画を表示できます。

    動画プロパティは、以下の各セクションにまとめられています。

    • ビデオ情報
    • 分類
    • ビデオファイル
    • メディア共有
    • オーディオトラック
    • 画像
    • ビデオビュー
    • カスタムフィールド
    • 空室状況
    • テキストトラック

    動画プロパティを編集するには、そのセクションの[編集]ボタンをクリックするか、そのセクションをダブルクリックします。一部のセクションでは、プロパティを直接編集できます（つまり、分類）。

    必須フィールド

    必須メタデータフィールドでは、お客様のVideo Cloudアカウントを設定できます。動画が非アクティブである場合、必須メタデータの一部を入力しなくても、メタデータを自由に編集し保存することができます。また、必須メタデータがすべて入力されるまで、動画はアクティブ化できません。動画がすでにアクティブな場合、編集中のセクションの変更を保存するには、そのセクションの必須メタデータをすべて入力するか、または動画を非アクティブ化する必要があります。必須フィールドの使用に関する詳細は、「動画の必須フィールドを設定する」を参照してください。

    動画情報

    [動画情報]セクションを使用して、基本的な動画プロパティを表示、編集できます。

    プロパティを編集するには、[編集]ボタンをクリックするか、そのセクションをダブルクリックします。以下のプロパティを編集できます。

    • 名前 -255文字以下です。
    • ID -システムによって割り当てられたビデオID（すべてのVideo Cloudアカウントで一意）。
    • 短い説明 -250文字以下です。
    • タグ - 1つ以上のタグ。複数指定する場合はコンマで区切ります。タグを使用して、スマートプレイリストに動画を追加したり、CMSAPIを使用して動画を検索したりできます。
      • タグは最大 128 文字です。128文字を超えるタグがある場合は、変更を保存しようとするとエラーが表示されます。
      • タグにはカンマを含めることはできません。たとえば、「屋根、屋根裏部屋 + 前庭」のタグはサポートされていません。
      • また、1本の動画に割り当て可能なタグは最大1,200個です。
      • 大文字と小文字のみが異なるタグ（小文字、大文字）は同じタグとして扱われ、大文字/小文字は自動的に元のタグのタグと一致します。
      • 大文字と小文字の区別に関する問題は、JavaScriptメソッドを使用して文字列を変更するなど、VideoCloudの外部で処理する必要があります。
      • タグを入力したら、コンマキーまたは Enter キーを押します。追加されたタグは、色付きのボックスに表示されます。
    • リファレンスID - パブリッシャーが動画またはプレイリストに割り当てる一意なID。150文字以内に制限されます。このリファレンスIDは動画IDやプレイリストIDとは別のものです。動画IDやプレイリストIDはVideo Cloudによって生成される一意なIDです。リファレンスIDを利用すると、Video Cloudと他のサービス（例: パブリッシャーのコンテンツ管理システム）を連係させることができます。なお、複数の動画に同じリファレンスIDを割り当てることができないため、リファレンスIDを一括編集することはできません。リファレンスIDは決して重複してはいけません。動画を削除すると、前に割り当てられていたリファレンスIDを再使用できます。
    • 動画プロジェクション - 動画がどのように記録されたかを示します。
      • 標準 - デフォルト。定期的なビデオ録画。
      • 360度 -ビデオが360度ビデオであることを示すためにチェックします。360の印が、Mediaモジュールの動画サムネイルに表示されます。360 度ビデオの操作の詳細については、「 360° ビデオの操作」を参照してください。
    • オフライン再生を有効にする -オンにすると、デバイス SDK を使用して作成したアプリケーションを使用するときに、ビデオをローカルに保存し、オフラインで表示できるようになります。このオプションは、アカウントがオフライン再生が有効になっている場合にのみ表示されます。
    • 関連リンク URL -http://または https://で始まる標準のウェブ URL。255 文字に制限されています。
    • 関連リンクテキスト -255文字に制限された関連リンクテキスト。
    • 詳細説明 -5000文字以下です。
    • 広告
      • 自由 -動画の再生時に広告は表示されません。
      • サポートされている広告 -動画の再生時に広告が表示される場合があります。
        注：お客様のアカウントで広告が有効ではないのに、「広告」プロパティを「広告サポート」に設定すると、変更を保存するときにエラーを受領します。
    • 広告キー - 広告サーバーURLの一部として渡されます。たとえば、section=blog および anotherKey=value1 を渡すには、section=blog&anotherKey=value1 と入力します。キー値ペアの追加については、「DoubleClick for Publishersヘルプ」を参照してください。広告キーを使用するとき、広告マクロを（プレーヤー広告プロパティの一部として設定された）広告サーバーURLに追加してください。広告マクロは、サーバーURLの変数をサポートします。IMA3プラグインが、変数を適切な値に置換します。詳細については、を参照してください。
    • 共有 - この動画が他のアカウントと共有されていることを示します。詳細については、を参照してください。メディアモジュールを使用したメディア共有。

    [保存]をクリックして、変更を保存します。

    分類

    [分類]セクションには、手動プレイリストビデオが割り当てられ、フォルダビデオはにあります。プロパティを編集するには、このセクションで直接変更を加えます。Enterキーを押して、変更を保存します。

    • 手動プレイリスト - この動画が属するプレイリスト。Xをクリックして、再生リストから動画を削除します。詳細は、「プレイリストを作成、管理する」を参照してください。
    • フォルダー - 動画を割り当てる先のフォルダー。詳細は、「フォルダーを使用して動画を整理する」を参照してください。

    Enterキーを押すと、変更が保存されます。

    動画ファイル

    [ビデオファイル]セクションには、ソースファイル名、使用されたトランスコーディングテクノロジ、および作成されたビデオレンディションの詳細が表示されます。また、以下が可能です。

    • ビデオソースファイルを置き換える
    • ビデオを再トランスコードする

    この例では、動的配信トランスコーディングが使用されました（ビデオがアップロードされたときに、動的配信またはコンテキストアウェアエンコーディングの取り込みプロファイルが使用されました）。

    注：アカウントでDRMが有効になっている場合は、DRMトグルスイッチもセクションに表示されることがあります。DRMの詳細については、を参照してください。 DRMによるコンテンツの取り込み。

    メディア共有

    ビデオがアフィリエイトアカウントに共有されている場合は、[メディア共有]セクションが表示され、ビデオが共有されているアフィリエイトが一覧表示されます。メディア共有の詳細については、を参照してください。メディアモジュールを使用したメディア共有。

    オーディオトラック

    動的配信またはコンテキストアウェアエンコーディングの取り込みプロファイルを使用してビデオがアップロードおよびトランスコードされた場合、[オーディオトラック]セクションが表示され、使用可能なオーディオレンディションに関する情報が含まれます。ビデオにオーディオトラックを追加する方法については、を参照してください。メディアモジュールを使用したビデオへのオーディオトラックの追加。

    注：ビデオがレガシー取り込みプロファイルを使用してトランスコードされた場合、[オーディオトラック]セクションは表示されません。

    画像

    [画像]セクションでは、動画のポスター画像およびサムネイル画像を表示できます。ポスター画像とサムネイル画像は、自動的に作成されます。中間点でソース動画からスナップショットが取得されます。取り込まれたデフォルトの画像が気に入らない場合、新しい画像をキャプチャーするか、または他の画像をアップロードして、ポスターおよびサムネイルとして使用できます。ポスター/サムネイル画像にカーソルを合わせ、表示アイコンをクリックします（）新しいブラウザタブで画像を開きます。

    動画視聴回数

    [ビデオビュー]セクションには、過去30日間のビデオビューの数が表示されます。詳細については、リンクをクリックしてパフォーマンスレポートまたはエンゲージメントレポートを開いてください。

    カスタムフィールド

    [カスタムフィールド]セクションを使用して、動画のカスタムフィールドを表示、編集できます。カスタムフィールド値を編集するには、[編集]ボタンをクリックするか、そのセクションをダブルクリックします。フィールド名の横にあるアスタリスク (*) は、フィールドが必須であることを示します。

    注：アカウントへのカスタムフィールドの追加についての詳細は、「カスタムメタデータフィールドを作成する」を参照してください。

    公開設定

    [公開設定]セクションは、動画の公開期間設定を表示します。公開期間の設定に関する詳細は、「動画の公開期間を設定する」を参照してください。

    テキストトラック

    [テキストトラック]セクションを使用して、動画にテキストトラックを追加できます。テキストトラックの構成の詳細については、を参照してください。メディアモジュールを使用してビデオにキャプションを追加する。
