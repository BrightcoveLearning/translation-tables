    360° ビデオの操作

    このトピックでは、VideoCloudで360度ビデオを操作する方法を学習します。

    360°ビデオは、あらゆる方向のビューが同時に記録されるビデオ記録です。360°ビデオは、全方位カメラまたはカメラのコレクションを使用して撮影されます。再生中、視聴者はパノラマのように視聴方向を制御できます。

    Video Cloudは、360°ビデオのアップロードと公開をサポートしています。360°ビデオのサンプルを以下に示します。ビデオを再生したら、マウスまたは矢印キーを使用してデスクトップでの表示を制御できます。モバイルデバイスで表示している場合はジャイロスコープを使用できます。

     

    ブラウザのサポートと360°/ VRビデオプラグインの既知の問題については、 360º/ VRビデオプラグイントピック。

    360°ビデオの録画

    360°ビデオは通常、複数のカメラの特別なリグを使用するか、複数のカメラレンズを含む専用カメラを使用して記録されます。結果として得られるフッテージは、1つのビデオを形成するためにステッチされます。360°ビデオを録画するために利用できる多くのカメラがあります。このトピックの360°ビデオは、Ricoh Theta Sを使用して記録されました。Brightcoveでは、ビデオを高解像度で記録することをお勧めします。

    アップロードする前に

    360°ビデオをVideoCloudにアップロードする前に、360°再生を有効にするためにビデオファイルに特定のメタデータを含める必要があります。Spatial Media Metadata Injectorアプリケーションを使用して、ファイルに360°ビデオが含まれていることを示すメタデータをビデオファイルに追加できます。360°メタデータを自動的に追加しないカメラで360°ビデオが録画された場合は、次の手順に従って追加します。

    1. の360°ビデオメタデータアプリをダウンロードするマックまたはウィンドウズ。
    2. 360°ビデオメタデータアプリをインストールして開きます。
    3. [開く]をクリックし、動画ファイルを選択します。
    4. [メタデータの挿入]をクリックします。
    5. 新しいファイルの名前を入力し、[保存]をクリックします。

    これで、ビデオをVideoCloudにアップロードする準備が整いました。

    360°ビデオのアップロード

    360°ビデオは、球の内部に投影されるように撮影されています。これは、プレーヤー内のビデオの表示領域は、ビデオ全体のほんの一部に過ぎないことを意味します。次のスクリーンショットは、この概念の大まかなアイデアを提供します。赤い長方形の内側の領域はプレーヤーに表示される領域ですが、これはビデオ全体のごく一部にすぎません。

    一度にプレーヤーに表示されるのはビデオのごく一部であるため、低品質のレンディションが表示される可能性がなく、非常に高品質のレンディションを再生する必要があります。本質的に、あなたはビデオの一部を「ズームイン」しており、レンディションの品質が悪いと、非常に質の低い視聴体験が得られます。ビデオをアップロードするとき、Brightcoveは、単一の1080pレンディションと少なくとも1つのオーディオレンディションを使用してカスタムDynamicDelivery取り込みプロファイルを作成することをお勧めします。

    注:カスタム取り込みプロファイルの作成については、「取り込みプロファイルの管理」を参照してください。モバイルデバイスには固定のビットレートと解像度の制限がある場合があるため、注意が必要です。一部の古いモバイルデバイスは、1080pを超える360°ビデオの再生に失敗する場合があります。

    360°ビデオのプロパティ

    すべてのVideoCloudビデオには、Video Cloud Studioを使用して、またはCMSAPIを使用してプログラムで設定できる一連のプロパティがあります。すべてのプロパティがVideoCloudStudioで公開されているわけではないことに注意してください。

    注：すべてのビデオプロパティのリストについては、を参照してください。 CMSAPIビデオフィールドリファレンス資料。

    ビデオが360°ビデオであることを示すには、メディアモジュールを使用してビデオのプロパティを編集し、ビデオプロジェクションプロパティに360°。

    注：ザ・CMS APIを設定するために使用することもできますprojectionプロパティに正距円筒図法。詳細については、を参照してください。 CMSAPIを使用した360°ビデオプロパティの設定資料。

    360°動画を公開する

    Brightcoveには、360°ビデオを公開する方法がいくつかあります。

    • Webページ上のBrightcoveプレーヤー
    • ブライトコーブソーシャル
    • BrightcoveネイティブプレーヤーSDK
    注：360°/ VRプラグインの最新バージョンを使用するには、BrightcovePlayerバージョン6.13.0以降を使用している必要があります。
    注：360°/ VRプラグインがiOSデバイスでビデオを正しくレンダリングするために、ビデオのプリロード再生前設定はに設定する必要がありますはいの一部としてプレーヤーのプロパティ。

    360°ビデオをWebページに公開する

    Brightcoveプレーヤーで360°ビデオを再生するには、プレーヤーに360°/ VRビデオプラグインが必要です。

    360°/ VRビデオプラグインを使用して新しいBrightcoveプレーヤーを作成するには、次の手順に従います。

    1. Players モジュールを開きます。
    2. 新しいプレーヤーを作成します。
    3. プレーヤー名をクリックして、プレーヤーのプロパティを開きます。
    4. 左側のナビゲーションで [ プラグイン ] をクリックします。
      注：このプラグインは、BrightcovePlayerバージョン6.xとのみ互換性があります。
    5. [ > プラグインカスタムプラグインを追加 ] をクリックします。
    6. のためにプラグイン名、 入るvr。
    7. JavaScript の URL には、次のように入力します。 
      //players.brightcove.net/videojs-vr/1/videojs-vr.min.js
    8. CSS の URL には、次のように入力します。 
      //players.brightcove.net/videojs-vr/1/videojs-vr.css
    9. 番号オプション（JSON）必要です。
    10. [ 保存 ]、[ プレーヤーを公開する ] の順にクリックします。

    これで、作成したばかりのプレーヤーを使用して360°ビデオを公開できます。

    BrightcoveSocialを使用した360°ビデオの公開

    Video Cloudアカウントにアップロードされた360°ビデオは、BrightcoveSocialを使用してソーシャルメディアアカウントに簡単に公開できます。ソーシャルモジュールを使用して360°ビデオを同期する場合、ビデオの取り込みに使用する取り込みプロファイルは、マスターファイルを保存するように構成する必要があります。そうしないと、ビデオは360°で同期されません。Video Cloudが360°ビデオを取り込むと、360°メタデータは結果のレンディションに保持されません。マスターなしで360°ビデオを同期する場合、最高品質のレンディションが同期で使用されます。レンディションには360°メタデータが含まれていないため、YouTube / Facebookはそれが360°ビデオであることを認識しません。

    Brightcove Socialを使用したビデオの公開については、を参照してください。ソーシャルメディアプラットフォームへのビデオの公開。

    Brightcove Native PlayerSDKを使用した360°ビデオの公開

    Brightcove Player SDKを使用して構築されたAndroidおよびiOSアプリは、360°ビデオを再生できます。SDKを使用して360°ビデオを公開する方法については、ネイティブSDKでの360°ビデオの使用資料。
