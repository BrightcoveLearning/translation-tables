    パフォーマンスレポートのリファレンス

    このトピックでは、Analyticsモジュールの一部として提供されるパフォーマンスレポートについて説明します。
     

    アナリティクスデータは、ネイティブプレーヤー SDK によって提供されるものを含め、ブライトコーブプレーヤーによって自動的に送信されます。Video Cloud 動画の配信に Brightcove Player を使用していない場合は、データコレクターにデータを送信するのに使用しているプレーヤーを計測する必要があります。概要を参照してください。データ収集API詳細については、トピックを参照してください。

    パフォーマンスレポートを表示するには、Analyticsモジュールを開き、左のナビゲーションから[パフォーマンス]を選択します。レポートには、選択した期間の1日ごとの動画表示回数を示すグラフが表示されます。ドロップダウンリストを使用して、別のメトリックを選択できます。

    • ビデオビュー -ストリームの開始時に記録された、ビデオの再生が開始された回数（巻き戻しまたは再生は含まれません）。それは個々の視聴者の尺度ではありません。プレロール広告のある動画の場合、動画の再生が開始されるまで視聴回数は記録されません
    • 動画表示回数 - 動画が読み込まれ、インタラクションの準備ができた回数
    • 再生率 - 読み込まれた動画が実際に再生された割合（再生回数/動画表示回数ン）
    • 視聴時間- 動画が視聴された時間（分）
    • エンゲージメントスコア - 動画表示1件当たりの平均視聴割合（全視聴割合の合計を動画表示回数で割って計算します）
    • 日次ユニーク視聴者 - 視聴者のIPアドレスとユーザーエージェントを使って計算されます。ブライトコーブはブラウザーによってクッキーがよりブロックされるようになっているため、サードパーティーのクッキーを使用しないことを決定しました。「日次ユニーク視聴者」が利用できるのは、アカウントレベルのみです。動画（など）を基準に絞り込むと、この指標は利用できません。さらに、ブライトコーブはユニーク視聴者をIPアドレスではなくお客様が提供した値に基づくようにする機能を提供しています。詳細は、開発者ドキュメントの「Analytics Event Tracking」を参照してください。
    • ビデオ帯域幅 -ビデオのみの帯域幅が含まれます。帯域幅データは3～5日遅れることにご注意ください。
    • 総帯域幅 -ビデオ、画像、プレーヤーの負荷の帯域幅が含まれます。帯域幅データは3～5日遅れることにご注意ください。

    アカウントでBrightcoveLiveが有効になっている場合は、すべてのビデオ、VODアセット、またはライブアセットのパフォーマンスデータを表示するオプションがあります。

    注
    • 解析レポートには、お客様のアカウントから削除された動画の解析が含まれます。削除された特定の動画に関するレポートを表示するために、削除された動画をAnalyticsモジュールで検索することはできません。そのようなレポートを取得するには、削除された動画の動画IDを知り、Analytics API を使用してレポートを生成する必要があります。
    • 毎日のユニークビューアのデータとビデオ帯域幅のデータが利用可能になるまで、最大3日かかります。
    • ビデオビューは、エンゲージメントレポートのx％でのビューと同じではありません。動画が初めて再生を開始すると、動画視聴回数がカウントされます。1%時点の視聴回数は、最初の1パーセントが視聴されると記録されます。ここでの違いは、最初のパーセントが複数回視聴された場合（たとえば、ビデオが自動再生に設定されている場合）、カウントが増加するのに対し、ビデオビューは、ビデオが最初に再生を開始したときにカウントされますが、カウントされないことです。リプレイ（ユーザーがページをリロードしたり、プレイリスト内の別のビデオを見たりしない限り）。
    • 同じアカウント内の同時レポートリクエストは連続して実行されます。レポートの日付範囲が大きく、多くのフィールドが選択されている場合は、すべての結果が返されるまで遅延が発生する可能性があります。大きなレポートのスケジュールを設定するのが良い方法です。
    • とページ内エクスペリエンス、ビデオがプレーヤーにロードされているように見えても、視聴者が再生ボタンのオーバーレイをクリックするまでビデオはロードされません。そのため、視聴者が動画を再生するまで Analytics モジュールでは、動画のインプレッションは記録されません。

    グラフの下にあるレポートスイッチャーを使用すると、別のレポートにすばやく切り替えることができます。

    レポートスイッチャーの右側にあるドロップダウンリストを使用して、レポートディメンションを選択できます。パフォーマンスレポートでは、次のディメンションを使用できます。

    • ビデオ
    • プレイヤー
    • トラフィックのソース
    • ドメインを表示
    • 国
    • デバイス
    • デバイスタイプ
    • デバイスOS
    • デバイスメーカー
    • ブラウザの種類
    • 日

    使用可能なディメンションは、選択したレポートに基づいて変わることに注意してください。レポートの下部で、表示する行数の選択や他のページへの移動ができます。

    前の期間と比較

    [前の期間と比較]チェックボックスを選択すると、前の期間のデータを表す点線グラフが追加されます。

    前の期間は、報告日範囲に現在選択されている範囲によって決まります。たとえば、過去7日間を日付範囲に選択している場合、前の期間は2週間前になります。週の傾向を比較する場合、週全体にわたる日付範囲の選択をすることが最適です。たとえば、開始日に月曜日を選択した場合は、終了日に日曜日を選択します。開始日が金曜日の場合は、終了日に木曜日を設定します。また、グラフの棒や点線の上にマウスポインターを置くと、日付とその日の値が表示されます。

    パフォーマンスレポートの詳細

    レポートに現れる最初の列は、選択されているレポート次元によって異なります（動画名、プレーヤー名、国、デバイスタイプなど）。すべてのレポートが以下のデータを表示します。

    • 動画表示回数 - 動画の再生が始まった回数。ストリームが開始されたときに記録されます（巻き戻しまたはリプレイを含みません）。個々の視聴者の数ではありません
    • 動画表示回数 - 動画がプレーヤに読み込まれインタラクションの準備ができた回数
    • 再生率 - 動画が読み込まれた後に動画を再生したユーザーの割合（動画表示回数を動画インプレッションで割った値）
    • 視聴時間 - 動画が表示された総時間（分）
    • エンゲージメントスコア - 動画表示1件当たりの平均視聴割合（全視聴割合の合計を動画表示回数で割って計算します）

    トラフィックソース

    トラフィックソース別にパフォーマンスを表示するとき、可能なトラフィックソースは次のとおりです。

    • 紹介 - ユーザーは動画へのリンクをたどった
    • 直接 - ユーザーはリンクを直接入力した、または紹介者が特定できなかった
    • 安全な検索 - ユーザーはGoogleアカウントにログインして検索を行い、結果をクリックした（有料またはオーガニック。この場合、検索情報は含まれない）
    • オーガニック検索 - ユーザーは検索を行って結果をクリックし、動画に行き着いた
    • 有料検索 - ユーザーは検索を行って有料結果をクリックし、動画に行き着いた

    レポートデータをエクスポートする

    レポートデータのダウンロードは、コンマ区切り形式とExcel形式で可能です。エクスポート機能は、Analytics APIコールからデータを直接保存することに注意してください。エクスポートされたデータはUIデータのようには操作されず、UIで表示されるフィールドのサブセットのみでなく、レポートの基礎となるデータすべてを含んでいます。UIに表示されるデータとエクスポートされたデータには以下のような違いがあります。

    • 単位の違い（分と秒）
    • 数値の四捨五入
    • データのエクスポートにはUIのどこにいるかにかかわらず、パフォーマンスとエンゲージメントのメトリックの両方が含まれます。
    • [その他/サードパーティー]項目は、関連する動画IDまたは動画名なしにエクスポートされます。

    すべてのデータをAPIの元のフォーマットで保持することで、UIで提供される方法以外でもデータを操作することができます。

    レポートは印刷に適したスタイルシートをサポートしています。したがって、レポートを印刷したりPDFに保存すると、印刷用のページが生成されます。

    動画解析を表示する

    動画次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。このレポートでは各プレーヤーに対する動画の名前、動画のID、長さ、およびパフォーマンスの詳細が表示されます。

    動画のリンクをクリックすると、その動画のパフォーマンスが表示されます。

    注：VideoCloudライブラリにないビデオがVideoCloudプレーヤー（リモートアセットなど）で再生される場合、その他/サードパーティ動画IDが関連付けられていない状態で表示されます。

    プレーヤー解析を表示する

    プレーヤー次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。レポートの本体では、各プレーヤーについてパフォーマンスの詳細が表示されます。

    プレーヤーのリンクをクリックすると、そのプレーヤーのパフォーマンスが表示されます。

    注：2016年2月以前は、Brightcove Player SDK for iOSまたはAndroidを使って構築されたアプリ内で使用されたプレーヤーは、[カスタムまたはネイティブ再生]というプレーヤー名で表示されていました。2016年2月から、これらのプレーヤーは、アプリで指定されるソースおよび配信先の値を使って識別できるようになりました。詳細は iOS または Android SDKのドキュメントを参照してください。

    トラフィックソースの解析を表示する

    トラフィックのソース次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。レポートの本文には、それぞれのパフォーマンスデータが表示されますトラフィックソース。選択した期間内にデータがあるカテゴリーのみが表示されます。

    トラフィックソースのリンクをクリックすると、選択したトラフィックソースに関するトラフィックソースパフォーマンスが表示されます。

    表示ドメイン解析を表示する

    表示ドメイン次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。このレポートには、表示ドメインに対するパフォーマンスデータが表示されます。表示ドメインとは、動画の視聴時に視聴者が使用していたドメインのことです。

    表示ドメインのいずれかのリンクをクリックすると、選択した表示ドメインの表示URLが表示されます。

    注：プレーヤーがiframeの場合、表示ドメインはiframeのページのURLとして記録されます。

    国の解析を表示する

    国次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。このレポートには、動画が視聴された各国のパフォーマンスデータが表示されます。選択した期間内にデータがある国のみが表示されます。

    注：Brightcoveは、サードパーティのサービスを使用して、IPアドレスと場所の情報を提供します。問題のIPアドレスのエントリがない場合、国の情報を取得できず、わからない現れる。

    いずれかの国のリンクをクリックすると、選択した国の地域または都市ごとのパフォーマンスが表示されます。アカウント、個々のビデオではありません。

    デバイス解析を表示する

    デバイス次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。このレポートには、動画の表示に使用された「デバイスタイプ - デバイスオペレーティングシステム」の組み合わせごとのパフォーマンスデータが表示されます。選択した期間内にデータがあるデバイスのみが表示されます。

    デバイスタイプのいずれかのリンクをクリックすると、そのデバイスのオペレーティングシステムのパフォーマンスが表示されます。

    注：Analyticsモジュールは、user-agentヘッダーを使用してデバイスを判断します。誰かがヘッダーを偽装してデバイスを使用している場合には、その偽装されたヘッダーに基づいてデータが作成されます。Analyticsモジュールでは、user-agentの改ざんを防ぐ対策を行いません。

    デバイスタイプ解析を表示する

    デバイスタイプ次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。レポートの本文には、パフォーマンスの詳細が表示されます。デバイスタイプ。選択した期間内にデータがあるカテゴリーのみが表示されます。

    デバイスタイプのいずれかのリンクをクリックすると、そのデバイスのオペレーティングシステムのパフォーマンスが表示されます。

    デバイスOS解析を表示する

    デバイスOS 次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。レポートの本文には、それぞれのパフォーマンスデータが表示されますデバイスのオペレーティングシステム。選択した期間内にデータがあるカテゴリーのみが表示されます。

    デバイスメーカー分析の表示

    いつデバイスメーカーディメンションが選択されている場合、グラフには、選択された期間の1日あたりのビデオビューの数が表示されます。レポートの本文には、それぞれのパフォーマンスデータが表示されますデバイスメーカー。選択した期間内にデータがあるデバイスのみが表示されます。

    ブラウザタイプ分析の表示

    いつブラウザの種類ディメンションが選択されている場合、グラフには、選択された期間の1日あたりのビデオビューの数が表示されます。レポートの本文には、それぞれのパフォーマンスデータが表示されますブラウザの種類。選択した期間内のデータを持つブラウザのみが表示されます。

    日解析を表示する

    日次元を選択すると、グラフには選択した期間の1日当たりの動画表示回数が表示されます。このレポートには、各日に対するパフォーマンスデータが表示されます。選択した期間内にデータがある日のみが表示されます。
