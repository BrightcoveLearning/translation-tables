    Trabajar con puntos de referencia en el módulo de medios

    En este tema aprenderá acerca de los puntos de referencia y cómo agregarlos, editarlos y eliminarlos mediante el módulo Media.

    Un punto de referencia es un marcador en un instante concreto a lo largo de un vídeo. Puede utilizar puntos de referencia para desencadenar cuñas publicitarias intermedias (es decir, que ocurren durante la reproducción del contenido del vídeo, como una pausa publicitaria) o para separar capítulos o escenas en un vídeo de formato largo.

    Hay dos tipos de puntos de referencia, pero ambos simplemente proporcionan un valor de cadena a la propiedad type del objeto de punto de referencia. El valor apropiado sólo es útil si se utiliza en JavaScript personalizado mientras se procesa la información del punto de referencia. 

    • Ad - Asigna el valor de cadena de AD a la propiedad type
    • Code - Asigna el valor de cadena de CODE a la propiedad type

    Los puntos de referencia se pueden agregar a vídeos mediante el módulo Media o mediante la API de catálogo. Para obtener más información sobre cómo agregar puntos de referencia mediante la API de catálogo, consulte el documento del desarrollador Implementación de puntos de referencia mediante programación.

    Adición de puntos de referencia

    Para agregar puntos de referencia a un vídeo, abra el módulo Media y siga estos pasos:

    1. Haga clic en el enlace del título del vídeo para abrir la página de propiedades del vídeo.
    2. Utilice el reproductor de vista previa para reproducir el vídeo. Detenga el vídeo en el punto en el que desee añadir un punto de referencia.
    3. Haga clic en Añadir punto de referencia.
    4. Si es necesario, ajuste los valores de código de tiempo (hh:mm:ss) para el punto de referencia. Tenga en cuenta que el campo segundos admite milisegundos después de la coma decimal.
    5. Seleccione el tipo de punto de referencia y, a continuación, agregue los metadatos apropiados.
      • ad - Se realizará una solicitud de anuncio. Al utilizar puntos de referencia publicitarios, el reproductor debe configurarse para la publicidad.
      • code - Los puntos de referencia se manejarán mediante programación. Consulte el documento del desarrollador Implementación de puntos de referencia mediante programación para obtener más información.
    6. Introduzca un nombre para el punto de referencia. Se puede acceder al valor del nombre mediante programación.
    7. Introduzca metadatos (para puntos de referencia de código) o pares clave/valor (para puntos de referencia de anuncio).
      • Metadatos :metadatos opcionales que se almacenan con el punto de referencia. Se puede acceder a los metadatos mediante programación. Un ejemplo de cuándo es posible que desee utilizar un punto de referencia de código y pasar metadatos es cuando desea mostrar diferentes llamadas a la acción mientras se está reproduciendo el vídeo. Consulte el ejemplo del reproductor de Brightcove: Puntos de referencia Mostrar CTA para un ejemplo.
      • Paresclave/valor: pares clave/valor que se pueden pasar a una solicitud de anuncio. Un par clave-valor es un conjunto de dos elementos de datos vinculados: una clave, que es un identificador único para algún elemento de datos, y el valor, que es los datos identificados o un puntero a la ubicación de esos datos. Los pares clave/valor se pueden utilizar para ayudar a proporcionar anuncios más segmentados. Lo que se introduzca aquí se copiará en la etiqueta de anuncio en la que se inserte la {mediainfo.ad_keys} macro. Para obtener más información, consulte Publicidad con el complemento IMA4.

    Después de agregar puntos de referencia a un vídeo, en tiempo de ejecución, los puntos de referencia se convierten en elementos de referencia WebVTT HTML5. Cuando se realiza esta conversión, algunos de los datos de punto de referencia, como el tipo y la hora de punto de referencia, se convierten en la pista de texto. El valor de tiempo de punto de referencia se asignará a la referencia WebVTT startTime y endTime las propiedades. Consulte este documento para obtener más información sobre la interfaz VTTcue.

    Edición de puntos de referencia

    Para editar puntos de referencia en un vídeo, abra el módulo Media y siga estos pasos:

    1. Haga clic en el enlace del título del vídeo para abrir la página de propiedades del vídeo.
    2. Haga clic en el punto de referencia para editarlo en la línea de tiempo.
      Nota: Una vez en el modo de edición, los botones Anterior y Siguiente se pueden utilizar para desplazarse a otros puntos de referencia.
    3. Edite las propiedades del punto de referencia y, a continuación, haga clic en Guardar.
    Nota: No es posible tener varios puntos de referencia con el mismo código de tiempo.

    Supresión de puntos de referencia

    Para eliminar puntos de referencia, abra el módulo Media y siga estos pasos:

    1. Haga clic en el enlace del título del vídeo para abrir la página de propiedades del vídeo.
    2. Haga clic en el punto de referencia que desea eliminar en la línea de tiempo.
    3. Haga clic en el icono Eliminar () situado junto a las propiedades del punto de referencia.
    4. Confirme la eliminación.
