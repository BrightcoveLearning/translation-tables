    Trabajar con vídeos de 360°

    En este tema aprenderá a trabajar con vídeos de 360 grados en Video Cloud.

    Los vídeos de 360° son grabaciones de vídeo en las que se graba una vista en todas las direcciones al mismo tiempo. Los vídeos de 360° se capturan utilizando una cámara omnidireccional o una colección de cámaras. Durante la reproducción, el visor tiene el control de la dirección de visualización como una panorámica.

    Video Cloud admite la carga y publicación de vídeos de 360°. A continuación se muestra un vídeo de 360° de muestra. Una vez que reproduzca el vídeo, puede usar el mouse o las teclas de flecha para controlar la vista en el escritorio, o el giroscopio si se ve en un dispositivo móvil.

     

    Para obtener información sobre la compatibilidad con el navegador y los problemas conocidos con el plugin de vídeo 360°/VR, consulte el tema del plugin de vídeo 360º/VR .

    Grabación de vídeos 360°

    Por lo general, el vídeo de 360° se graba utilizando un equipo especial de varias cámaras o usando una cámara dedicada que contiene varios objetivos de cámara. El material de archivo resultante se cose para formar un solo video. Hay muchas cámaras disponibles para grabar vídeo de 360°. El vídeo 360° de este tema fue grabado utilizando un Ricoh Theta S. Brightcove recomienda grabar su vídeo a alta resolución.

    Antes de subir

    Antes de cargar un vídeo de 360° a Video Cloud, el archivo de vídeo debe incluir determinados metadatos para poder activar la reproducción en 360°. La aplicación Spatial Media Metadata Injector se puede utilizar para agregar metadatos a un archivo de vídeo que indica que el archivo contiene vídeo de 360°. Si el vídeo de 360° se grabó en una cámara que no agrega automáticamente los metadatos de 360°, siga estos pasos para agregarlo:

    1. Descargue la aplicación Metadatos de vídeo 360° para Mac o Windows.
    2. Instale y, a continuación, abra la aplicación Metadatos de vídeo 360°.
    3. Haga clic en Abrir y seleccione el archivo de vídeo.
    4. Haga clic en Inyectar metadatos.
    5. Introduzca un nombre para el nuevo archivo y haga clic en Guardar.

    El vídeo ya está listo para ser cargado en Video Cloud.

    Subiendo vídeos en 360°

    El vídeo de 360° se filma para ser proyectado en el interior de una esfera. Esto significa que el área visible del vídeo en el reproductor es solo una fracción del vídeo completo. La siguiente captura de pantalla le da una idea aproximada de este concepto. El área dentro del rectángulo rojo es lo que es visible en el reproductor, pero es sólo una pequeña parte de todo el vídeo.

    Dado que solo una pequeña parte del vídeo estará en el reproductor a la vez, desea que se reproduzca una representación de muy alta calidad, sin posibilidad de que aparezca una representación de menor calidad. En esencia, usted está «ampliando» parte del vídeo y una representación de mala calidad produciría una experiencia de visualización de muy mala calidad. Al cargar su video, Brightcove recomienda crear un perfil de ingesta de Entrega dinámica personalizado con una única reproducción de 1080p y al menos una reproducción de audio.

    Nota: Para obtener información sobre cómo crear perfiles de ingesta personalizados, consulte Gestión de perfiles de ingesta. Se debe tener cuidado, ya que los dispositivos móviles a veces tienen límites fijos de velocidad de bits y resolución. Es posible que algunos dispositivos móviles antiguos no reproduzcan vídeos de 360° superiores a 1080p.

    Propiedades de vídeo de 360°

    Todos los vídeos de Video Cloud tienen un conjunto de propiedades que se pueden establecer mediante Video Cloud Studio o mediante programación mediante la CMS API. Tenga en cuenta que no todas las propiedades están expuestas en Video Cloud Studio.

    Nota: Para obtener una lista de todas las propiedades de vídeo, consulte el documento de referencia de campos de vídeo de la CMS API .

    Para indicar que un vídeo es un vídeo de 360°, utilice el módulo Media para editar las propiedades del vídeo y establezca la propiedad Proyección de vídeo en 360°.

    Nota: El también se CMS API puede utilizar para establecer la projection propiedad en equirectangular. Para obtener más información, consulte Configuración de las propiedades de vídeo de 360° con el documento de la CMS API .

    Publicación de vídeos en 360°

    Brightcove ofrece varias formas de publicar vídeos en 360°:

    • Brightcove Players en una página web
    • Brightcove Social
    • SDK de reproductor nativo de Brightcove
    Nota: Para utilizar la versión más reciente del plugin 360°/VR, debe utilizar Brightcove Player versión 6.13.0 o posterior.
    Nota: Para que el plugin 360°/VR renderice los vídeos correctamente en dispositivos iOS, el ajuste Precargar vídeo antes de la reproducción debe establecerse en Sí como parte de las propiedades del reproductor.

    Publicación de vídeos 360° en páginas web

    Para reproducir vídeos de 360° en un Brightcove Player, el reproductor necesita el plugin de vídeo de 360°/VR.

    Para crear un nuevo Brightcove Player con el plugin de vídeo 360°/VR, siga estos pasos:

    1. Abra el módulo Players.
    2. Crea un nuevo jugador.
    3. Haga clic en el nombre del reproductor para abrir las propiedades del reproductor.
    4. Haga clic en Plugins en la navegación izquierda.
      Nota: Este plugin es compatible únicamente con Brightcove Player versión 6.x.
    5. Haga clic en Agregar un complemento > personalizado.
    6. Para el nombre del complemento, escriba vr.
    7. Para la URL de JavaScript, escriba: 
      //players.brightcove.net/videojs-vr/1/videojs-vr.min.js
    8. Para la URL CSS, escriba: 
      //players.brightcove.net/videojs-vr/1/videojs-vr.css
    9. No se necesitan opciones (JSON) .
    10. Haga clic en Guardar y, a continuación, Publicar el reproductor.

    Ahora puede publicar el vídeo de 360° con el reproductor que acaba de crear.

    Publicación de vídeos 360° con Brightcove Social

    Los vídeos de 360° cargados en su cuenta de Video Cloud se pueden publicar fácilmente en sus cuentas de redes sociales mediante Brightcove Social. Tenga en cuenta que al utilizar el módulo Social para sincronizar un vídeo de 360°, el perfil de ingesta utilizado para ingerir el vídeo debe estar configurado para guardar el archivo maestro o el vídeo no se sincronizará en 360°. Cuando Video Cloud ingiere vídeos de 360°, los metadatos de 360° no se conservan en las representaciones resultantes. Al sincronizar un vídeo de 360° sin un maestro, se utiliza la representación de la más alta calidad en la sincronización. Como la representación no incluye los metadatos de 360°, YouTube/Facebook no sabrá que es un video de 360°.

    Para obtener información sobre cómo publicar vídeos mediante Brightcove Social, consulte Publicación de vídeos en plataformas de medios sociales.

    Publicación de vídeos 360° con los SDK de Brightcove Native Player

    Las aplicaciones Android e iOS creadas con los SDK de Brightcove Player pueden reproducir vídeos de 360°. Para obtener información sobre la publicación de vídeos de 360° con los SDK, consulte el documento Uso de vídeos de 360° con los SDK nativos .
