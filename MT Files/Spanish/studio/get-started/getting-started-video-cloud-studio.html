    Introducción a Video Cloud Studio

    En este tema se proporciona información general sobre la interfaz y los módulos de Video Cloud Studio. Video Cloud Studio le permite administrar el flujo de trabajo de publicación de vídeo en equipos de escritorio, tabletas y dispositivos móviles.
     

    Para acceder a Video Cloud Studio, vaya a: https://studio.brightcove.com. Inicie sesión con su dirección de correo electrónico y su contraseña. La página principal proporciona la capacidad de:

    • Ver notificaciones del sistema
    • Administrar la configuración de Video Cloud
    • Seleccionar una cuenta y ver el perfil de usuario
    • Acceder a módulos de Video Cloud
    • Ver el panel de control de Video Cloud
    Nota: El tiempo de espera de la sesión de Video Cloud Studio es de 24 horas.

    Visualización de notificaciones del sistema

    En el encabezado de Video Cloud Studio aparecerá un icono de campana cuando el sistema le haya enviado alguna notificación. Para mostrar la lista de notificaciones del sistema, haga clic en este icono.

    El encabezado de Video Cloud Studio mostrará un icono de advertencia cuando haya problemas con el sistema. Al hacer clic en el icono se abrirá la página Estado del sistema Brightcove.

    Administrar la configuración

    Se puede acceder al menú de configuración de Video Cloud Studio desde el menú ADMIN . El menú ADMIN de Video Cloud proporciona acceso a:

    • Administrar usuarios - Administrar los usuarios autorizados a utilizar Video Cloud Studio
    • Información de cuenta : ver información sobre la cuenta actual de Video Cloud
    • Configuración de carga : proporciona opciones para personalizar el módulo Cargar
    • Ingerir perfiles - Administrar los perfiles de ingesta en su cuenta de Video Cloud
    • Campos de vídeo : marque los campos de vídeo estándar como sea necesario y también vea, cree y modifique campos de metadatos personalizados para la cuenta
    • Compartir contenido multimedia - Compartir vídeos con otras cuentas
    • Configuración del reproductor : proporciona la opción de utilizar la autenticación básica de nombre de usuario y contraseña para acceder a las API del sistema de entrega y administración de reproductores
    • Configuración en vivo - Administrar los ajustes utilizados por el módulo Live
    • Autenticación de API - Crear y administrar credenciales de API
    • Facturación - Ver información de facturación de la cuenta
    Nota: Para poder acceder al menú ADMIN se precisan permisos de administrador de la cuenta.

    Selección de una cuenta

    Si dispone de varias cuentas de Video Cloud, utilice el selector de cuentas de Video Cloud para seleccionar la cuenta a la que desee acceder. Para cambiar de cuenta, haga clic en su nombre - cuenta en la esquina superior derecha de Video Cloud Studio. La cuenta activa en cada momento se indica mediante un punto delante del nombre de la misma.

    Haga clic en el enlace Mi perfil para actualizar la información de usuario y la contraseña. Para obtener más información, consulte Gestión de información de perfil.

    Acceso a módulos

    El menú de Video Cloud Studio proporciona acceso a los módulos de Studio. Acceda al menú haciendo clic en INICIO.

    Tenga en cuenta que los módulos que se muestran se basan en la suscripción a Video Cloud y el perfil de usuario.

    • Subir - Subir vídeos a su cuenta de Video Cloud
    • Medios - Establecer propiedades de vídeo, organizar vídeos mediante carpetas y publicar vídeos
    • Analytics - Ver análisis sobre el rendimiento de vídeo y reproductor
    • Jugadores - Crear y dar estilo a los reproductores y generar código de publicación para los jugadores
    • Live - Crear y transmitir eventos en vivo
    • Playout en la nube - Crea canales lineales en vivo para videos
    • Gallery - Crea portales de vídeo para tu contenido de vídeo
    • Social - Distribuir vídeos a cuentas de redes sociales
    • Audiencia : conecta el análisis de vídeo de Video Cloud a las plataformas de automatización de marketing

    Visualización del tablero

    La página principal de Video Cloud Studio muestra el panel de control de Video Cloud. El panel contiene una serie de «tarjetas» que proporcionan métricas de alto nivel en los vídeos de tu cuenta. Las siguientes tarjetas están disponibles:

    • Vistas de vídeo : muestra el número de vistas de vídeo por día durante el período de tiempo seleccionado. Al hacer clic en el enlace Ver detalles de la vista de vídeo se abrirá el informe de rendimiento, donde podrá ver el rendimiento del vídeo.
    • Resumen de la cuenta : muestra el número total de vídeos y reproductores de la cuenta y el número de vídeos creados y actualizados recientemente. Al hacer clic en un número se abrirá el módulo Media o Players.
    • Tipo de dispositivo : muestra el tipo de dispositivos utilizados para ver vídeos durante el período de tiempo seleccionado. Al hacer clic en el enlace Ver vistas de vídeo por tipo de dispositivo se abrirá el informe de rendimiento, donde podrá ver el rendimiento del vídeo por tipo de dispositivo.
    • ¿Qué hay de nuevo? - Muestra información de actualización de Video Cloud Enterpriseduct. Para obtener una lista completa de actualizaciones, consulte las Notas de la versión de Video Cloud Studio.
    • Vídeos de tendencia superior : muestra vídeos con la mayor cantidad de vistas durante el período de tiempo seleccionado. Al hacer clic en el enlace del título de la tarjeta se abrirá el Informe de rendimiento, donde podrá ver el rendimiento del vídeo.
    • Visualización de regiones : muestra las ubicaciones del visor para el período de tiempo seleccionado. Al hacer clic en Ver vistas de vídeo por país , se abrirá el informe de rendimiento, donde podrá ver las vistas de vídeo por país.

    Localización

    El idioma de Video Cloud Studio se puede cambiar cambiando el idioma del perfil. Se admiten los siguientes idiomas:

    • Inglés
    • Español (España)
    • Japonés
    • Francés (Francia)
    • Chino (tradicional)
    • Coreano

    Para obtener más información, consulte Gestión de información de perfil.
