    Publicación de vídeos y listas de reproducción con el módulo multimedia

    En este tema aprenderá a publicar vídeos y listas de reproducción mediante el módulo Media.

    Este tema contiene información sobre cómo publicar vídeos y listas de reproducción en la Web. Para obtener información sobre las otras opciones de publicación, consulte:

    • Publicación de un vídeo en correo electrónico
    • Publicación de vídeos en plataformas de redes sociales
    • Publicación de vídeos en una experiencia en página

    Si desea asignar dinámicamente contenido de vídeo a un reproductor, obtenga el código de inserción del reproductor y, a continuación, utilice una de las técnicas disponibles para asignar vídeos dinámicamente a un reproductor.

     
     
    Nota: Para obtener información sobre cómo crear listas de reproducción, consulte Creación y administración de listas de reproducción.

    Publicar vídeos y listas de reproducción

    Para publicar un vídeo o una lista de reproducción, siga estos pasos.

    1. Abra el módulo Media haciendo clic en Media en el encabezado de navegación.
    2. Para publicar un solo vídeo, haga clic en Todos los vídeos en el panel de navegación lateral y, a continuación, realice una de las acciones siguientes:
      • Haga clic en la casilla de verificación situada junto al vídeo y, a continuación, haga clic en Publicar e incrustar > Web Player
      • Haga clic en el nombre del vídeo y, a continuación, haga clic en Publicar e incrustar > Web Player en la página de propiedades del vídeo
    3. Para publicar una lista de reproducción, haga clic en Todas las listas de reproducción en el panel de navegación lateral y, a continuación, realice una de las acciones siguientes:
      • Haga clic en la casilla de verificación situada junto a la lista de reproducción y, a continuación, haga clic en Publicar e incrustar Reproductor > Web
      • Haga clic en el nombre de la lista de reproducción y luego haga clic en Publicar lista de reproducción en la página de vídeos de la lista de reproducción
    4. Compruebe si se abre el cuadro de diálogo de publicación. En el título del cuadro de diálogo de publicación aparecerá el nombre del vídeo o de la lista de reproducción.

      Problema conocido: Debido a que el reproductor de previsualización se carga en Studio a través de HTTPS, si su contenido se retransmite a través del protocolo HTTP, el reproductor de previsualización no podrá reproducir el contenido y el navegador mostrará un mensaje error avisando de una secuencia de comandos insegura. No se trata de un error de programación de Studio, sino de una limitación de seguridad del navegador.

    5. Seleccione el reproductor que desee utilizar. Al publicar listas de reproducción, hay algunas propiedades del reproductor relacionadas con la lista de reproducción que se pueden configurar para controlar la apariencia y el comportamiento de la lista de reproducción. Consulte la sección Configuración de propiedades de la lista de reproducción a continuación para obtener más detalles.

      La vista previa se actualizará para mostrar el reproductor seleccionado.

    6. (Opcional) El enlace a la vista previa situado bajo el reproductor de previsualización permite ver una vista previa del reproductor en otra pestaña del navegador. Haga clic en el botón Acortar para generar una URL acortada.
    7. Elija una opción para definir el tamaño del reproductor:
      • Responsive - El tamaño del reproductor se ajustará automáticamente en función del tamaño del navegador
      • Corregido - El tamaño del jugador se basaba en los parámetros de ancho y alto
    8. Seleccione una relación de aspecto para el jugador.
      • 16:9 o 4:3 - introduzca el ancho del jugador (la altura se determinará automáticamente)
      • Personalizado : para introducir manualmente el ancho y la altura del jugador
    9. Indique la Anchura del reproductor y la Altura del reproductor.
    10. Seleccione las Unidades de medida.
    11. Seleccione el tipo de código de publicación que desea copiar haciendo clic en Estándar o Avanzado.
      • Estándar - Código de inserción iframe - Recomendado para la mayoría de los casos. Este tipo coloca el reproductor dentro de una etiqueta <iframe>. Si se especifican las dimensiones del reproductor, el tamaño del iframe se adaptará a ellas. De lo contrario, el iframe podrá colocarse dentro de una etiqueta <div> y, gracias a su capacidad de reacción, el reproductor cambiará de tamaño para ocupar todo el espacio de <div>.
        Nota: La anchura del iframe debe ser superior a 640 píxeles para que la lista de reproducción aparezca.
      • Avanzado - Código de inserción en la página - Permite que el reproductor exista directamente en la página HTML, no en un iframe. De este modo, los desarrolladores disfrutan de las ventajas de acceder al reproductor y a las propiedades y eventos asociados a él. Al utilizar el código de incrustación en la página se pierde la facilidad de uso del reproductor como unidad sencilla e independiente para aplicaciones y compartición en redes sociales. Esta opción solo está disponible en reproductores publicados.
      Para obtener más información sobre los tipos de código de publicación, consulte Elegir el código incrustado correcto.
    12. Copie el código de publicación. Cuando haya terminado, haga clic en Cerrar.

    Agregar un ID de aplicación al código incrustado

    En el pasado, los editores solían crear nuevos reproductores con nombres diferentes para utilizarlos en las distintas secciones de un sitio o en varios sitios. De este modo, cuando se revisaban los análisis de los reproductores resultaba más sencillo ver cuáles presentaban un mayor número de visionados. Al agregar un parámetro de ID de aplicación al código de inserción, se puede utilizar el mismo reproductor en un sitio o sitios, y el ID de aplicación se puede utilizar para identificar dónde se publicó el reproductor. El uso de un identificador de aplicación puede ayudarle a reducir el número de reproductores que tiene que crear y mantener. Para obtener información sobre cómo agregar un ID de aplicación al código de incrustación, consulte Adición de un ID de aplicación al código de inserción del reproductor.

    Establecer propiedades de lista de reproducción

    Existen algunas propiedades del reproductor relacionadas con listas de reproducción que se pueden configurar cuando se trabaja con listas de reproducción. Para configurar las propiedades del reproductor relacionadas con la lista de reproducción, siga estos pasos.

    1. Abra el módulo Players.
    2. Haga clic en el nombre del reproductor para acceder a las propiedades del mismo.
    3. Como parte de las propiedades Información del jugador , seleccione un Tipo de jugador de lista de reproducción. Haga clic en Guardar.
    4. Haga clic en Estilo en el menú de navegación de la izquierda.
    5. Seleccione un tipo de lista de reproducción:
      • Lista de reproducción (Vertical) - La lista de reproducción se muestra en el lado derecho del reproductor.
      • Lista de reproducción (Horizontal) - Lista de reproducción se muestra en la parte inferior del reproductor.
      • Lista de reproducción (oculta) : no se muestra la lista de reproducción.
    6. Cuando se selecciona un tipo de lista de reproducción como parte de las propiedades Información del reproductor , se pueden configurar propiedades de reproducción adicionales:
      • Orden de reproducción aleatoria - Los vídeos se reproducirán en orden aleatorio
      • Reproducir vídeo cuando se selecciona - Hacer clic en una miniatura en el menú de la lista de reproducción hará que el vídeo se reproduzca cuando se cargue, incluso si el vídeo anterior en el reproductor estaba en pausa.
      • Modo de reproducción continua : los vídeos avanzarán automáticamente y se repetirán indefinidamente
      • Cuenta atrás de vídeo: sólo se activa cuando se marca el modo de reproducción continua
        • Hasta la siguiente carta (avance automático) - Se muestra una carta de arriba siguiente en el jugador
        • Arriba siguiente pantalla final (cuenta atrás) - Se muestra una cuenta atrás de la pantalla final en el reproductor
    7. Para guardar los cambios, haga clic en Guardar. Recuerde que debe publicar el reproductor para que los cambios se reflejen en el reproductor publicado.

    Cuando se utiliza el código de inserción Estándar (iframe) , la lista de reproducción tiene un botón mostrar/ocultar y se ajusta automáticamente. Cuando utilice el código de inserción Avanzado (en página) , debe insertar la lista de reproducción como un <div> elemento HTML y escribir su propio CSS para posicionar y ajustar el tamaño de la lista de reproducción. Esto le da un control total sobre la visualización de la lista de reproducción. En este último caso, la lista de reproducción ni siquiera necesita ser adyacente al jugador si no quieres que lo sea. los Reproductor avanzado (incrustado en la página) sección de la Implementar listas de reproducción El documento tiene un contenido sustancial sobre el formato y el estilo de una lista de reproducción.
