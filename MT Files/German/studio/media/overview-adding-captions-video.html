    Übersicht: Hinzufügen von Untertiteln zu einem Video

    In diesem Thema erfahren Sie, wie Sie mit Untertiteln in Video Cloud arbeiten.

    Einführung

    Fügen Sie Videos Untertitel hinzu, um dem Betrachter auf Wunsch zusätzliche Informationen oder eine Verständnishilfe zu bieten. Untertitel sind erforderlich, damit taube und hörbehinderte Betrachter den Audio-Teil von Videos verstehen.

    Ein solcher Text kann für die Barrierefreiheit verwendet werden, um Übersetzungen der Audiospur oder einen anderen Zweck bereitzustellen, aber die zugrunde liegende Technologie ist dieselbe.

    Speicherort der Untertiteldatei

    Sie können auswählen, wo die WebVTT-Untertiteldatei gehostet werden soll:

    1. An einer Remote-URL (Remote-Asset) - In diesem Fall wird die WebVTT-Datei auf Ihrem Server gehostet und Sie fügen den Video-Metadaten die URL für die Datei hinzu.
      Hinweis: Wenn Sie Untertiteldateien auf Ihrem eigenen Server hosten, stellen Sie sicher, dass sich die Datei unter einer URL befindet, auf die der Brightcove Player zugreifen kann. Auf dem Server, der die Untertiteldatei hostet, sollte auch Cross-Origin Resource Sharing (CORS) aktiviert sein. Weitere Informationen über CORS finden Sie hier.
    2. Hochgeladen mit Video Cloud (aufgenommen ) - In diesem Fall wird die WebVTT-Datei in die Video Cloud hochgeladen (aufgenommen). Dies kann über das Medienmodul oder über die APIs erfolgen.
    Hinweis: Wenn in die Videoquelldatei, die Sie in Video Cloud aufnehmen, 608/708 Untertitel eingebettet sind, werden diese während der Verarbeitung des Videos ignoriert nicht in den Videowiedergaben vorhanden. Um Untertitel mit Ihren Videos zu haben, müssen Sie diese über die hinzufügen Medienmodul , das Dynamic Ingest API , oder der CMS-API (für remote gehostete Untertitel)

    Untertitel mit einem Video verknüpfen

    Es gibt verschiedene Möglichkeiten, eine Untertiteldatei einem Video zuzuordnen:

    1. Verwenden von Studio
    2. Ändern des Spieler-Publishing-Codes
    3. Verwenden der APIs

    Hinzufügen von Untertiteln in Studio

    Untertitel können mithilfe des Medienmoduls einem Video zugeordnet werden. Weitere Informationen finden Sie unter Hinzufügen von Untertiteln zu Videos mithilfe des Medienmoduls.

    Ändern des Spieler-Publishing-Codes

    In diesem Fall a <track> Das Tag wird dem Player-Veröffentlichungscode hinzugefügt. Weitere Informationen finden Sie unter Programmgesteuertes Hinzufügen von Untertiteln zu Videos.

    Hinzufügen von Untertiteln über die APIs

    Wie Sie Untertitel über die APIs hinzufügen, hängt davon ab, ob die Untertiteldatei aufgenommen oder als Remote-Asset verwendet wird.

    • Remote-Asset-Untertitel - Wenn Sie die Untertitel lieber selbst hosten möchten (oder einen Untertitelanbieter haben, der sie hostet), können Sie die Untertitel mithilfe der CMS-API einem Video zuordnen. Weitere Informationen finden Sie unter Verwenden der CMS-API: Fügen Sie WebVTT-Untertitel hinzu.

    • Fügen Sie der Video Cloud Untertitel hinzu - Wenn Brightcove Ihre Untertitel hosten soll, können Sie Ihre WebVTT-Datei mithilfe der Dynamic Ingest-API aufnehmen. Weitere Informationen finden Sie unter WebVTT-Dateien aufnehmen.

    Hinweis: Unabhängig von der Methode zum Hinzufügen von Untertiteldateien zu einem Video dürfen nicht mehr als 1000 Textspuren hinzugefügt werden.

    Untertitel entfernen

    Untertitel können mit dem Medienmodul oder über die CMS-API aus Videos entfernt werden. Öffnen Sie in Studio das Medienmodul und bearbeiten Sie die Videoeigenschaften. Bearbeiten Sie die Textspuren Eigentum nach Bedarf.

    Beschriftungen können auch mit dem entfernt werden CMS API durch ein Video aktualisieren Anfrage und Übermittlung eines Anfragekörpers mit einem leeren text_tracks Array (oder ein Array, aus dem das diesem Satz von Untertiteln zugeordnete Objekt entfernt wurde).

    Untertiteldesign

    Das Styling kann mithilfe von CSS auf WebVTT-Beschriftungen angewendet werden. Sehen Bildunterschriften stylen für Details.
