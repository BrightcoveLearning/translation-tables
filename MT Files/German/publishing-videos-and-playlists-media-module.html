    Veröffentlichen von Videos und Playlists mit dem Medienmodul

    In diesem Thema erfahren Sie, wie Sie Videos und Wiedergabelisten mithilfe des Medienmoduls veröffentlichen.

    Dieses Thema enthält Informationen zum Veröffentlichen von Videos und Wiedergabelisten im Web. Informationen zu den anderen Veröffentlichungsoptionen finden Sie unter:

    • Veröffentlichen eines Videos per E-Mail
    • Veröffentlichen von Videos auf Social Media-Plattformen
    • Veröffentlichen von Videos zu einem In-Page-Erlebnis

    Wenn Sie einem Player dynamisch Videoinhalte zuweisen möchten, rufen Sie die Code für den Player einbetten und verwenden Sie dann eine der verfügbaren Techniken Weisen Sie einem Player dynamisch Videos zu.

     
     
    Hinweis: Informationen zum Erstellen von Wiedergabelisten finden Sie unter Erstellen und Verwalten von Wiedergabelisten.

    Veröffentlichen von Videos und Wiedergabelisten

    Führen Sie die folgenden Schritte aus, um ein Video oder eine Wiedergabeliste zu veröffentlichen.

    1. Klicken Sie im Navigationsheader auf Media, um das Modul Media zu öffnen.
    2. Klicken Sie auf, um ein einzelnes Video zu veröffentlichen Alle Videos im seitlichen Navigationsbereich und führen Sie dann einen der folgenden Schritte aus:
      • Aktivieren Sie das Kontrollkästchen neben dem Video und klicken Sie dann auf Veröffentlichen und einbetten> Web Player
      • Klicken Sie auf den Videonamen und dann auf Veröffentlichen und einbetten> Web Player auf der Seite mit den Videoeigenschaften
    3. Klicken Sie auf, um eine Wiedergabeliste zu veröffentlichen Alle Wiedergabelisten im seitlichen Navigationsbereich und führen Sie dann einen der folgenden Schritte aus:
      • Aktivieren Sie das Kontrollkästchen neben der Wiedergabeliste und klicken Sie dann auf Veröffentlichen und einbetten> Web Player
      • Klicken Sie auf den Namen der Wiedergabeliste und dann auf Playlist veröffentlichen auf der Playlist-Videoseite
    4. Bestätigen Sie, dass das Dialogfeld für die Veröffentlichung geöffnet wird. Der Name des veröffentlichten Videos oder der Wiedergabeliste erscheint im Titel des Dialogfelds für die Veröffentlichung.

      Bekanntes Problem: Da der Vorschau-Player in Studio über HTTPS geladen wird, spielt er Inhalt, der über HTTP bereitgestellt wird, nicht ab, und im Browser erscheint eine Meldung mit einem Hinweis auf unsichere Skripts. Dies ist kein Studio-Fehler, sondern eine Einschränkung der Browsersicherheit.

    5. Wählen Sie einen Player. Beim Veröffentlichen von Wiedergabelisten gibt es einige Wiedergabelisten-bezogene Player-Eigenschaften, die zur Steuerung festgelegt werden können Aussehen und Verhalten der Wiedergabeliste. Siehe die Festlegen der Eigenschaften von Wiedergabelisten Abschnitt unten für Details.

      Die Vorschau wird mit dem ausgewählten Player aktualisiert.

    6. (Optional) Verwenden Sie den Vorschaulink unter dem Vorschau-Player, um eine Vorschau vom Player in einem anderen Browserfenster anzuzeigen. Drücke den Verkürzen Schaltfläche, um eine verkürzte URL zu generieren.
    7. Wählen Sie eine Option für die Player-Größe:
      • Responsive – die Player-Größe wird basierend auf der Browsergröße automatisch angepasst.
      • Fixed – die Player-Größe basiert auf den Parametern für die Breite und Höhe.
    8. Wählen Sie ein Seitenverhältnis für den Player.
      • 16: 9 oder 4: 3 - Geben Sie eine Spielerbreite ein (die Höhe wird automatisch ermittelt).
      • Benutzerdefiniert - um die Breite und Höhe des Players manuell einzugeben
    9. Geben Sie unter Player Width und Player Height die Player-Breite und -höhe ein.
    10. Wählen Sie unter Units die Maßeinheit.
    11. Wählen Sie die Art des zu kopierenden Veröffentlichungscodes aus, indem Sie auf klicken Standard oder Fortgeschrittene.
      • Standard - Iframe-Einbettungscode - In den meisten Fällen empfohlen. Bei diesem Einbettungscode wird der Player in einem <iframe>-Tag platziert. Falls Player-Abmessungen angegeben sind, wird die Größe des iframes an diese angepasst. Sind keine Player-Abmessungen angegeben, kann der iframe in einem <div>-Tag platziert werden. Da der Player anpassungsfähig ist, wird seine Größe an die gesamte Fläche des <div> angepasst.
        Hinweis: Die Iframe-Breite muss größer als 640 Pixel sein, damit die Wiedergabeliste angezeigt wird.
      • Erweitert - In-Page-Einbettungscode - Ermöglicht dem Player, direkt auf der HTML-Seite zu existieren, nicht in einem Iframe. So haben Entwickler einfachen Zugriff auf den Player und die entsprechenden Eigenschaften und Ereignisse. Bei Verwendung des Codes für die Einbettung in die Seite können Sie den Player nicht als einzelne integrierte Einheit in Social-Media-Anwendungen und zum Teilen in sozialen Netzwerken verwenden. Diese Option ist nur für den veröffentlichten Player verfügbar.
      Weitere Informationen zu den Arten von Veröffentlichungscode finden Sie unter Auswahl der richtigen Einbettung Code.
    12. Kopieren Sie den Veröffentlichungscode. Wenn Sie fertig sind, klicken Sie auf Schließen.

    Hinzufügen einer Anwendungs-ID zur Einbettung Code

    In der Vergangenheit wurden von Publishern häufig neue Player mit anderen Namen erstellt, die auf anderen Abschnitten einer Website oder auf anderen Websites verwendet wurden. Beim Überprüfen von Player-Analytics lässt sich anhand verschiedener Player-Namen leichter feststellen, mit welchen Playern die meisten Ansichten erfolgten. Durch Hinzufügen eines Anwendungs-ID-Parameters zur Einbettung Code kann derselbe Player für eine Site oder Sites verwendet werden, und die Anwendungs-ID kann verwendet werden, um zu identifizieren, wo der Player veröffentlicht wurde. Durch die Verwendung einer Player-Anwendungs-ID reduziert sich gegebenenfalls die Anzahl von Playern, die erstellt und gepflegt werden muss. Informationen zum Hinzufügen einer Anwendungs-ID zur Einbettung Code, siehe Hinzufügen einer Anwendungs-ID zum Player Embed Code.

    Festlegen der Eigenschaften von Wiedergabelisten

    Es gibt einige Wiedergabelisten-bezogene Player-Eigenschaften, die beim Arbeiten mit Wiedergabelisten konfiguriert werden können. So konfigurieren Sie die Wiedergabelisten-bezogenen Player-Eigenschaften: folge diesen Schritten.

    1. Öffnen Sie das Modul Players.
    2. Klicken Sie auf einen Player, um die Player-Eigenschaften zu öffnen.
    3. Im Rahmen des Spielerinformationen Eigenschaften, wählen Sie a Spielertyp von Wiedergabeliste. Klicken Sie auf Save.
    4. Klicken Sie im linken Navigationsmenü auf Styling .
    5. Wählen Sie einen Playlist-Typaus:
      • Playlist (Vertikal) - Die Playlist wird auf der rechten Seite des Players angezeigt.
      • Wiedergabeliste (Horizontal) — Die Playlist wird unten im Player angezeigt.
      • Wiedergabeliste (versteckt) - Die Wiedergabeliste wird nicht angezeigt.
    6. Wenn ein Spielertyp von Wiedergabeliste wird als Teil des ausgewählt Spielerinformationen Eigenschaften, zusätzlich Wiedergabe Eigenschaften können konfiguriert werden:
      • Randomize Play-Reihenfolge - Videos werden in einer zufälligen Reihenfolge abgespielt
      • Video abspielen, wenn ausgewählt - Wenn Sie auf ein Miniaturbild im Wiedergabelistenmenü klicken, wird das Video beim Laden abgespielt, selbst wenn das vorherige Video im Player pausiert wurde
      • Kontinuierlicher Spielmodus - Videos werden automatisch vorankommen und sich auf unbestimmte Zeit wiederholen
      • Video-Countdown - Nur aktiviert, wenn der Continue-Play-Modus aktiviert ist
        • Nächste Karte hoch (automatischer Vorschuss) - Eine nach oben nächste Karte wird im Player angezeigt
        • Auf demnächsten Endbildschirm (Countdown) - Ein Countdown für den Endbildschirm wird im Player angezeigt
    7. Klicken Sie auf Save, um die Änderungen zu speichern. Veröffentlichen Sie den Player, damit die Änderungen im veröffentlichten Player berücksichtigt werden.

    Bei Verwendung der Standard (Iframe) Code einbetten, die Wiedergabeliste verfügt über eine Schaltfläche zum Ein- und Ausblenden und wird automatisch in der Größe und Position positioniert. Bei Verwendung der Erweitert (In-Page) Code einbetten, müssen Sie die Wiedergabeliste als HTML einfügen <div> Element und schreiben Sie Ihr eigenes CSS, um die Wiedergabeliste zu positionieren und zu dimensionieren. Dadurch haben Sie die volle Kontrolle über die Anzeige der Wiedergabeliste. Im letzteren Fall muss sich die Wiedergabeliste nicht einmal neben dem Player befinden, wenn Sie dies nicht möchten. Das Erweiterter Player (Einbetten in die Seite) Abschnitt der Implementieren von Wiedergabelisten Das Dokument enthält umfangreiche Inhalte zum Formatieren und Gestalten einer Wiedergabeliste.
